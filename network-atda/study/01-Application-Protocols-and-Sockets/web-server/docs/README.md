이 디렉터리는 `Web Server`를 공부할 때 반복해서 다시 볼 개념과 판단 근거만 남깁니다. 문제 이해의 출발점은 아니며, `problem/README.md`와 `python/README.md`를 읽은 뒤 필요한 문서만 다시 참조합니다.

## 먼저 볼 파일
- [`content-types.md`](concepts/content-types.md)
- [`error-handling.md`](concepts/error-handling.md)

- 기준 검증: `make -C study/01-Application-Protocols-and-Sockets/web-server/problem test`
- 개념 문서 위치: `docs/concepts/`
- 참고 자료 위치: `docs/references/`

- TCP 소켓과 `HTTP/1.1` 응답 조합으로 정적 파일 서버를 구현하는 파일럿 과제입니다.
