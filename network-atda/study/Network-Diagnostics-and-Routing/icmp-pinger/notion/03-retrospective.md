# 03 회고

## 이번 프로젝트에서 확실히 얻은 것
- 운영체제 명령 뒤에 숨겨져 있던 ICMP 패킷 구조와 raw socket 권한 모델을 직접 이해했다.
- 네트워크 진단 도구도 결국 정확한 바이트 파싱과 timeout 설계 위에서 돌아간다는 점을 배웠다.

## 아직 약한 부분
- root 권한이 필요한 점 때문에 실행 환경 재현성이 떨어질 수 있다.
- IPv6와 platform-specific 차이를 아직 다루지 않았다.

## 포트폴리오로 키우려면
- 권한 부족 시 친절한 가이드를 출력하고 JSON 결과 저장을 추가한다.
- `ping` 비교 결과와 packet capture 이미지를 함께 넣어 학습 내용을 시각화한다.
