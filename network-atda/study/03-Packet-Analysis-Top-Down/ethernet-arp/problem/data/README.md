# Ethernet & ARP Lab — Trace Data

This directory contains pre-captured Wireshark trace files for the Ethernet & ARP lab.

> Note: This repository ships reproducible **synthetic traces** generated by
> `Wireshark-Labs/scripts/generate_synthetic_traces.py` so labs can run without
> privileged packet capture.

## Trace Files

| File | Description |
| :--- | :--- |
| `ethernet-arp.pcapng` | ARP request/reply exchanges and general Ethernet framing on a LAN |

## Source

These traces are based on the Wireshark Ethernet/ARP lab from the companion website of *Computer Networking: A Top-Down Approach* (8th Edition) by Kurose & Ross.

To generate your own traces:
1. Clear ARP cache: `sudo arp -a -d` (macOS) or `sudo ip neigh flush all` (Linux)
2. Start Wireshark capture on your LAN interface
3. Access a local host or browse the web (triggers ARP for default gateway)
4. Stop the capture and save as `.pcapng`
