# Notion Documents for SMTP Client

이 디렉터리는 SMTP Client 과제의 노션 업로드용 문서를 담고 있습니다.

- `essay.md`: SMTP 프로토콜 이해, 구현 과정, 테스트 전략 등을 서술한 블로그형 에세이입니다.
- `timeline.md`: CLI 명령어 및 개발 과정을 순차적으로 기술한 타임라인 문서입니다.

**목적별 읽기:**

- 개념적 설명이나 코드 리뷰가 필요하면 `essay.md`를 참조하세요.
- 구체적인 명령어, 실행 절차가 필요하면 `timeline.md`를 확인하세요.
