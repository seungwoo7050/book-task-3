# TCP & UDP Lab — Trace Data

This directory contains pre-captured Wireshark trace files for the TCP & UDP lab.

> Note: This repository ships reproducible **synthetic traces** generated by
> `Wireshark-Labs/scripts/generate_synthetic_traces.py` so labs can run without
> privileged packet capture.

## Trace Files

| File | Description |
| :--- | :--- |
| `tcp-upload.pcapng` | A file uploaded to a server via HTTP POST over a TCP connection |
| `udp-dns.pcapng` | DNS queries and responses over UDP |

## Source

These traces are based on the Wireshark TCP/UDP lab from the companion website of *Computer Networking: A Top-Down Approach* (8th Edition) by Kurose & Ross.

To generate your own traces:
1. Open Wireshark and start a capture
2. For TCP: upload a file to a web server or use `curl -X POST`
3. For UDP: run DNS queries with `nslookup` or `dig`
4. Stop the capture and save as `.pcapng`
