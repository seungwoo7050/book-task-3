# HTTP Lab — Trace Data

This directory contains pre-captured Wireshark trace files for the HTTP lab.

> Note: This repository ships reproducible **synthetic traces** generated by
> `Wireshark-Labs/scripts/generate_synthetic_traces.py` so labs can run without
> privileged packet capture.

## Trace Files

| File | Description |
| :--- | :--- |
| `http-basic.pcapng` | Basic HTTP GET request and 200 OK response |
| `http-conditional.pcapng` | Conditional GET with If-Modified-Since and 304 response |
| `http-long-document.pcapng` | Large HTML document transfer across multiple TCP segments |
| `http-embedded-objects.pcapng` | Web page with multiple embedded images |

## Source

These traces are based on the Wireshark HTTP lab from the companion website of *Computer Networking: A Top-Down Approach* (8th Edition) by Kurose & Ross.

To generate your own traces:
1. Open Wireshark and start a capture on your active network interface
2. Navigate to the corresponding URL in your browser
3. Stop the capture and save as `.pcapng`
