# DNS Lab — Trace Data

This directory contains pre-captured Wireshark trace files for the DNS lab.

> Note: This repository ships reproducible **synthetic traces** generated by
> `Wireshark-Labs/scripts/generate_synthetic_traces.py` so labs can run without
> privileged packet capture.

## Trace Files

| File | Description |
| :--- | :--- |
| `dns-nslookup.pcapng` | DNS queries generated by nslookup for various domains and record types |
| `dns-web-browsing.pcapng` | DNS queries triggered by browsing to web pages |

## Source

These traces are based on the Wireshark DNS lab from the companion website of *Computer Networking: A Top-Down Approach* (8th Edition) by Kurose & Ross.

To generate your own traces:
1. Open Wireshark and start a capture on your active network interface
2. Run `nslookup` commands or browse to websites
3. Stop the capture and save as `.pcapng`
