# IP & ICMP Lab — Trace Data

This directory contains pre-captured Wireshark trace files for the IP & ICMP lab.

> Note: This repository ships reproducible **synthetic traces** generated by
> `Wireshark-Labs/scripts/generate_synthetic_traces.py` so labs can run without
> privileged packet capture.

## Trace Files

| File | Description |
| :--- | :--- |
| `ip-traceroute.pcapng` | ICMP packets from a traceroute session showing TTL expiry |
| `ip-fragmentation.pcapng` | Large ping packets triggering IP fragmentation |

## Source

These traces are based on the Wireshark IP lab from the companion website of *Computer Networking: A Top-Down Approach* (8th Edition) by Kurose & Ross.

To generate your own traces:
1. Start Wireshark capture
2. For traceroute: run `traceroute <host>` (Linux/Mac) or `tracert <host>` (Windows)
3. For fragmentation: run `ping -s 3500 <host>` (large payload to force fragmentation)
4. Stop the capture and save as `.pcapng`
