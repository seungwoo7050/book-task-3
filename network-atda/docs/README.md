# Repository Docs

이 디렉터리는 저장소 전체에 공통으로 적용되는 문서만 모아 둔 곳입니다. 개별 프로젝트의 구현 메모와 개념 문서는 각 `study/<stage>/<project>/` 아래에서 읽습니다.

- [`readme-contract.md`](readme-contract.md): 루트, 단계, 프로젝트, 하위 README가 반드시 드러내야 할 공개 표면 계약
- [`curriculum-map.md`](curriculum-map.md): 어떤 단계 순서로 공부하면 좋은지와 단계 간 연결 고리
- [`project-set-audit.md`](project-set-audit.md): 현재 프로젝트 집합의 강점, 의도적인 공백, 확장 판단 기준
- [`repository-inventory.md`](repository-inventory.md): 현재 저장소 구조와 각 디렉터리 역할
- [`project-template.md`](project-template.md): 새 프로젝트를 추가할 때 맞춰야 할 기본 구조와 README 규칙
- [`verification-matrix.md`](verification-matrix.md): canonical 검증 명령과 현재 상태
