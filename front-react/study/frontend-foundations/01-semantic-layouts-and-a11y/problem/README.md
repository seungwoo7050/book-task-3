# Product Brief

프로비넌스: `authored`

## 목표

설정/대시보드 성격의 UI shell을 만들면서 semantic layout, form grouping, help/error text pairing, keyboard reachable interaction을 확인한다.

## 포함 범위

- semantic landmarks
- responsive two-column to single-column layout
- labeled forms
- inline help and error pairing
- keyboard focus states

## 제외 범위

- 복잡한 데이터 fetching
- local persistence
- 실제 라우팅
