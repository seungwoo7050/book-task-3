# Product Brief

프로비넌스: `authored`

## 목표

directory/explorer 형태의 UI를 만들면서 request state와 navigation state를 연결한다.

## 포함 범위

- loading, empty, error, retry
- abort and request race
- mock API latency
- query-param driven navigation
- integration and E2E smoke

## 제외 범위

- 실제 서버 운영
- authentication
- server-side rendering
