# Product Brief

프로비넌스: `authored`

## 목표

task/workspace board를 만들면서 selection, filter, sort, edit, URL state, local persistence를 연결한다.

## 포함 범위

- filter and sort controls
- row selection and inline edit
- localStorage persistence
- URL query sync
- keyboard interaction

## 제외 범위

- 실제 network request
- complex authentication
- server cache
