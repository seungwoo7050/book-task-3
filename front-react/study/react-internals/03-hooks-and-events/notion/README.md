# Notion Notes — hooks-and-events

이 폴더는 `03-hooks-and-events` 패키지에 대한 개발 노트입니다. 각각의 파일은 다음 용도로 쓰입니다.

1. **00-problem-framing.md** – 이 단계에서 해결하려는 문제와 이전 패키지와의 연계.
2. **01-approach-log.md** – 실행한 작업과 설계 결정의 시간 순 기록.
3. **02-debug-log.md** – Encountered bugs and how they were debugged.
4. **03-retrospective.md** – 회고 및 향후 개선 방향.
5. **04-knowledge-index.md** – 주요 타입, 함수, 외부/내부 의존 관계.
6. **05-development-timeline.md** – 워크스페이스 설치, 테스트, 빌드 명령어 등 개발 여정.
