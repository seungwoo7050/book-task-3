# Notion Notes — vdom-foundations

`01-vdom-foundations`의 개발 노트입니다. 구성은 일관된 6개 파일로 되어 있으며, 각 파일의 역할은 다음과 같습니다.

1. **00-problem-framing.md** – 왜 이 라이브러리를 만들었는지, 해결하고자 하는 문제점.
2. **01-approach-log.md** – 구현 여정, 설계 선택과 코드 패턴.
3. **02-debug-log.md** – 실제 버그와 그 원인 및 해결 방법.
4. **03-retrospective.md** – 회고, 성과와 배운 점.
5. **04-knowledge-index.md** – 핵심 타입/함수 목록과 관계.
6. **05-development-timeline.md** – 개발 도구, 커맨드, 빌드/테스트 흐름.

**사용 팁**
- 처음 접할 때는 번호 순서대로 읽으면 전체 흐름이 자연스럽게 드러납니다.
- 특정 부분만 빠르게 검토하고 싶다면 관련 번호의 문서를 열어보세요.
- 이 패키지를 다른 프로젝트(02, 03 등)에서 참조할 때는 knowledge index와 development timeline이 특히 유용합니다.
