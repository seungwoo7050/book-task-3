# Notion Notes — render-pipeline

`02-render-pipeline` 프로젝트의 개발 노트 모음입니다. 6개의 문서를 통해 단계별로 내용을 파악할 수 있습니다.

1. **00-problem-framing.md** – 동기와 목표, 이전 패키지와의 관계.
2. **01-approach-log.md** – 구현 로그, 방식, 중요한 API 설명.
3. **02-debug-log.md** – 발생한 오류 사례와 수정 기록.
4. **03-retrospective.md** – 프로젝트 전체를 돌아보는 회고.
5. **04-knowledge-index.md** – 핵심 개념, 내부 타입, 외부 의존성.
6. **05-development-timeline.md** – 워크스페이스 설정, dependency 링크, 테스트/빌드 명령어.

**읽는 방법**
- 전체적인 플로우를 알고 싶다면 처음부터 순서대로.
- vdom-foundations 의존 문제나 패키지 연결 방식을 찾고 있다면 02-debug-log와 05-development-timeline이 도움이 됩니다.
- 이후 `03-hooks-and-events`로 넘어갈 때는 knowledge index와 retrospective를 참고하세요.

노트는 로컬에서만 유지되며 자유롭게 편집할 수 있습니다.
