# Notion Notes — Client Onboarding Portal

이 디렉터리에는 `02-client-onboarding-portal` 프로젝트를 설명하는 로컬 노트가 들어 있습니다. 각 문서의 목적은 다음과 같습니다.

1. **00-problem-framing.md** – 문제 정의와 이 프로젝트가 다루는 영역.
2. **01-approach-log.md** – 개발 과정과 중요 설계 결정.
3. **02-debug-log.md** – 디버깅 기록과 문제 해결 사례.
4. **03-retrospective.md** – 프로젝트 회고, 장단점.
5. **04-knowledge-index.md** – 개념, 패턴, 다른 프로젝트와의 연관.
6. **05-development-timeline.md** – 사용한 커맨드, 도구 버전, 단계별 기록.

**읽는 법**
- 처음 접하는 독자는 00부터 순서대로 읽으면 전체 흐름을 빠르게 파악할 수 있습니다.
- 인증, draft save, wizard 흐름 등 개별 주제를 검토하려면 해당 번호의 문서를 열어보세요.
- 발표나 리뷰 준비 시에는 retrospective와 development-timeline 문서가 도움이 됩니다.

이 폴더는 `.gitignore` 대상이므로 로컬에서 자유롭게 수정하고 갱신할 수 있습니다.
