# Notion Notes — Realtime Collab Workspace

이 디렉터리에는 `03-realtime-collab-workspace` 프로젝트를 설명하는 로컬 노트가 들어 있습니다. 각 문서의 목적은 다음과 같습니다.

1. **00-problem-framing.md** – 문제 정의와 이 프로젝트가 다루는 경계.
2. **01-approach-log.md** – 구현 과정과 중요 설계 결정.
3. **02-debug-log.md** – 디버깅 기록과 문제 해결 사례.
4. **03-retrospective.md** – 프로젝트 회고와 다음 확장 후보.
5. **04-knowledge-index.md** – 핵심 개념, 타입, 테스트 앵커 정리.
6. **05-development-timeline.md** – 도구, verify 흐름, 단계별 메모.

**읽는 법**
- 처음 접하는 독자는 00부터 순서대로 읽으면 전체 흐름을 빠르게 파악할 수 있습니다.
- patch 모델, reconnect replay, conflict surface 같은 주제를 보려면 01과 02를 먼저 읽으면 됩니다.
- 발표나 리뷰 준비 시에는 03과 05가 가장 직접적으로 도움이 됩니다.

이 폴더는 `.gitignore` 대상이므로 로컬에서 자유롭게 수정하고 갱신할 수 있습니다.
