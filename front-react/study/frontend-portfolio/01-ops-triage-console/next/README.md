# Next App

이 디렉터리는 `Ops Triage Console`의 실제 실행 가능한 앱과 테스트 코드를 담는다.

## 디렉터리

- `app/`: Next App Router 엔트리와 라우트
- `src/`: 컴포넌트, hooks, mock service, state helpers
- `public/`: 정적 자산
- `tests/`: unit, integration, E2E 테스트

## 실행

프로젝트 루트 패키지 스크립트를 통해 실행한다.

```bash
cd study
npm run dev:portfolio
```
