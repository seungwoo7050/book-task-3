# Repository Docs

이 디렉터리는 `front-react` 전체에 적용되는 공통 규칙과 재설계 근거를 모은다.

## 문서 목록

- [curriculum-map.md](curriculum-map.md): 전체 3트랙 9프로젝트 핵심 경로와 순서
- [junior-skill-matrix.md](junior-skill-matrix.md): 주니어 끝자락 기준 역량과 프로젝트 매핑
- [legacy-audit.md](legacy-audit.md): 레거시 트리 감사 결과와 재설계 판단 근거
- [notion-readiness-audit.md](notion-readiness-audit.md): 로컬 `notion/` 품질 기준과 공개/보류 분류
- [verification-policy.md](verification-policy.md): 설치, 테스트, 타입체크, 상태 판정 기준
- [project-template.md](project-template.md): 프로젝트 구조, 상태 용어, 프로비넌스 규칙

## 사용 원칙

- 저장소 설명은 현재 실제 경로를 기준으로 쓴다.
- 레거시 자료는 참고 근거로만 사용하고 활성 작업 경로처럼 안내하지 않는다.
- tracked 문서는 공개 인덱스와 재현 절차 중심으로 유지하고, 과정 로그는 `notion/`으로 분리한다.
