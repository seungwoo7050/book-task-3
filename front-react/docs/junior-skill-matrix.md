# Junior-End Skill Matrix

이 문서는 이 저장소가 "주니어 끝자락" 수준에서 어떤 역량을 커버하려는지, 그리고 각 프로젝트가 그 역량의 어떤 증거가 되는지 정리한다.

## 역량 축

| 역량 | 설명 | 주 프로젝트 |
| --- | --- | --- |
| HTML/CSS/a11y | semantic structure, responsive layout, form labeling, keyboard navigation | `frontend-foundations/01-semantic-layouts-and-a11y` |
| DOM/event/async | DOM state sync, event handling, delegation, local persistence, async UI state | `frontend-foundations/02-dom-state-and-events`, `frontend-foundations/03-networked-ui-patterns` |
| React mental model | VDOM, diff, render/commit, hooks, events, runtime limits | `react-internals/*` |
| 제품형 UI | data-heavy internal tool, customer-facing onboarding flow, route/form/error UX | `frontend-portfolio/*` |
| 테스트/품질 | unit, integration, E2E, a11y smoke, deterministic demo scenarios | `react-internals/01-vdom-foundations`, `frontend-portfolio/01-ops-triage-console` |
| 배포/문서화 | 실행 가능한 README, presentation doc, case study, 검증 스크립트 | 전체 track 공통 |

## 이 저장소가 커버하지 않는 것

- 미들급 범위의 시스템 설계
- 대규모 협업 동기화
- production backend 운영
- 복잡한 infra / observability 운영

## 읽는 법

- breadth를 보고 싶으면 `frontend-foundations`와 `frontend-portfolio`를 먼저 본다.
- framework depth를 보고 싶으면 `react-internals`를 본다.
- 채용용 결과물을 보고 싶으면 `frontend-portfolio`를 본다.
