# Infobank 경력기술서

## 한 줄 소개

과제 요구를 공식 제출 답과 확장 답, 검증, 발표 자료까지 닫힌 결과물로 정리하는 경험을 갖고 있습니다.

## 공통 코어

- 42서울 정규과정 수료
- `ft_transcendence`의 Django 백엔드, 42 OAuth, JWT, TOTP 기반 2FA 경험
- `cs-core`, `network-atda`, `database-systems` 기반 이해

## 대표 경험

### 1. MCP 추천 최적화 capstone

- release candidate CRUD, compatibility gate, release gate, artifact export까지 정리

### 2. 챗봇 상담 품질 관리 capstone

- baseline/candidate 비교, guardrail, evidence verification, dashboard proof 구성

### 3. backend-common

- FastAPI 기준선으로 인증/인가, API 경계, 검증 구조를 공통 기반으로 정리

## 기술 스택

- TypeScript, Python, FastAPI, React
- evaluation artifacts, compare proof, release gate

## 검증 습관

- 공식 답과 확장 답을 분리하고, 제출 기준 버전을 front door 문서에 명시합니다.

## 성장 방향

요구사항을 빠르게 제품화 관점으로 재구성하고, proof와 검증 산출물까지 함께 닫는 개발자로 성장하고 싶습니다.
