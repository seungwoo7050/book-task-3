# Node Backend 경력기술서

## 한 줄 소개

NestJS 기반 서비스에서 인증, 저장소, 캐시, Swagger, Docker Compose 실행 경로를 함께 정리하는 Node.js 백엔드 개발 경험을 갖고 있습니다.

## 공통 코어

- 42서울 정규과정 수료
- `ft_transcendence`의 Django 백엔드, 42 OAuth, JWT, TOTP 기반 2FA 경험
- `cs-core`, `network-atda`, `database-systems` 기반 이해

## 대표 경험

### 1. 10 shippable-backend-service

- 채용 제출용 표면으로 다시 패키징한 대표 결과물
- build, test, e2e, run 경로를 README 기준으로 재현 가능하게 구성

### 2. Node Backend Architecture 트랙

- bridge, core, applied 구조로 런타임부터 서비스 표면까지 단계적으로 학습

### 3. backend-common

- 인증/인가, API 경계, 서비스 분해 감각을 FastAPI 기준선으로 강화

## 기술 스택

- Node.js, TypeScript, NestJS
- PostgreSQL, Redis, Swagger, Docker Compose

## 검증 습관

- `pnpm run build`, `pnpm run test`, `pnpm run test:e2e`를 canonical 검증 경로로 유지합니다.

## 성장 방향

Node.js 환경에서도 서비스 표면과 운영성 기준을 함께 설명하는 백엔드 엔지니어로 성장하고 싶습니다.
