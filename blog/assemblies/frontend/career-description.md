# Frontend 경력기술서

## 한 줄 소개

React, TypeScript, Next.js 기반으로 제품형 UI를 구현하고, 브라우저 런타임 특성과 검증 자동화까지 함께 고려하는 프론트엔드 개발자입니다.

## 공통 코어

- 42서울 정규과정 수료
- `ft_transcendence`에서 Django 백엔드, 42 OAuth, JWT, TOTP 기반 2FA 경험 확보
- `cs-core`, `network-atda`, `database-systems`를 통해 시스템 이해 기반 강화

## 대표 경험

### 1. mini-vrew

- 프론트엔드 단독 개발 및 Vercel 배포
- FFmpeg WASM, WebGL, Web Audio API, IndexedDB를 제품형 UX에 연결

### 2. front-react 포트폴리오 트랙

- `Ops Triage Console`, `Client Onboarding Portal` 중심
- 운영 화면과 고객-facing 플로우를 각각 다른 품질 기준으로 설계

### 3. 프론트엔드 바깥 구조 이해

- React internals, 모바일, 공통 코어 프로젝트를 통해 UI 뒤의 구조까지 함께 학습

## 기술 스택

- React, Next.js, TypeScript, Vite
- TanStack Query, React Hook Form, Zod
- Playwright, Vitest
- FFmpeg WASM, WebGL, Web Audio API, IndexedDB

## 검증 습관

- UI도 `typecheck + unit test + E2E`를 함께 묶습니다.
- 화면과 함께 재검증 가능한 명령을 README 앞단에 둡니다.

## 성장 방향

실무 제품에서 더 큰 사용자 흐름과 협업 구조를 경험하며, 프론트엔드 품질 기준과 구조를 함께 만드는 개발자로 성장하고 싶습니다.
