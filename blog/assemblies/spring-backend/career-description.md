# Spring Backend 경력기술서

## 한 줄 소개

Spring Boot 기반 서비스에서 인증, JPA 도메인 모델링, Redis/Kafka 연동, Testcontainers 검증까지 함께 정리한 백엔드 개발자입니다.

## 공통 코어

- 42서울 정규과정 수료
- `ft_transcendence`의 Django 백엔드, 42 OAuth, JWT, TOTP 기반 2FA 경험
- `cs-core`, `network-atda`, `database-systems` 기반 이해

## 대표 경험

### 1. commerce-backend-v2

- 대표 포트폴리오 capstone
- modular monolith, JPA + Flyway, Redis, Kafka, Testcontainers 기반 구조 정리

### 2. Federation / Authorization / Messaging 랩

- 인증 강화, RBAC, 이벤트 경계를 랩 단위로 분리 학습
- 이후 capstone에서 다시 통합

### 3. backend-common

- FastAPI 기준선에서 인증/인가, 서비스 경계, async notification 감각 강화

## 기술 스택

- Spring Boot, JPA, Flyway, Redis, Kafka
- MockMvc, JUnit, Testcontainers

## 검증 습관

- `make lint`, `make test`, `make smoke`, Compose health 확인을 canonical 기준으로 사용합니다.

## 성장 방향

도메인과 운영 기준을 함께 설명하는 Spring 백엔드 엔지니어로 더 깊어지고 싶습니다.
