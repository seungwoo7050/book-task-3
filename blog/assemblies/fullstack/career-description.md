# Fullstack 경력기술서

## 한 줄 소개

사용자 흐름을 화면, 인증, API, 데이터, 검증 체계까지 하나의 제품 문제로 연결해 설계하는 풀스택 개발자입니다.

## 공통 코어

- 42서울 정규과정 수료
- `ft_transcendence`의 Django 백엔드, 42 OAuth, JWT, TOTP 기반 2FA 경험
- `cs-core`, `network-atda`, `database-systems` 기반 이해

## 대표 경험

### 1. mini-vrew

- 단독 개발 및 Vercel 배포
- 사용자 경험과 처리 파이프라인을 함께 설계

### 2. Client Onboarding Portal

- sign-in부터 submit retry까지 이어지는 사용자 흐름 설계

### 3. workspace-backend / workspace-backend-v2-msa

- 인증, 협업형 도메인, 알림을 API와 비동기 경계로 연결

## 기술 스택

- React, Next.js, TypeScript
- FastAPI, PostgreSQL, Redis, Docker

## 검증 습관

- 프론트엔드는 typecheck, unit test, E2E를 묶고, 백엔드는 smoke와 verification report를 함께 둡니다.

## 성장 방향

화면과 API를 함께 믿고 맡길 수 있는 개발자로 성장하고, 장기적으로는 제품 흐름 전체의 설계 품질을 높이는 풀스택 엔지니어가 되고 싶습니다.
