# Go Backend 경력기술서

## 한 줄 소개

Go 기반 API에서 권한 경계, 데이터 모델, 비동기 처리, 재현 가능한 검증 경로를 함께 설계하는 백엔드 개발자입니다.

## 공통 코어

- 42서울 정규과정 수료
- `ft_transcendence`의 Django 백엔드, 42 OAuth, JWT, TOTP 기반 2FA 경험
- `cs-core`, `network-atda`, `database-systems` 기반 이해

## 대표 경험

### 1. 18 Workspace SaaS API

- Go 기반 대표 포트폴리오 과제
- JWT auth, RBAC, invitation, issue/comment workflow, notification 통합

### 2. backend-common

- FastAPI 기준선으로 인증/인가, API 경계, async notification, verification report 정리

### 3. Event / Transaction 트랙

- `15 Event Pipeline`, `17 Game Store Capstone`을 통해 outbox, idempotency, transaction 경계 반복 학습

## 기술 스택

- Go, PostgreSQL, Redis, JWT, OpenAPI
- outbox, worker, smoke/e2e

## 검증 습관

- `go test`, `make e2e`, `make smoke`를 canonical 경로로 유지합니다.
- 문서와 구현 drift를 줄이기 위해 OpenAPI와 smoke 시나리오를 함께 맞춥니다.

## 성장 방향

실무에서 더 큰 도메인과 운영 복잡도를 경험하며, Go 기반 서비스 구조와 운영 기준을 함께 설계하는 백엔드 엔지니어로 성장하고 싶습니다.
