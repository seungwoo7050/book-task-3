# Go Backend 경력기술서 Module

## 한 줄 소개

Go 기반 API에서 권한 경계, 데이터 모델, 비동기 처리, 재현 가능한 검증 경로를 함께 설계하는 백엔드 개발자입니다.

## 핵심 역량

- JWT auth / refresh
- 조직 단위 RBAC / invitation
- worker 분리와 outbox 기반 notification
- `go test`, `make e2e`, `make smoke` 기반 재현성

## 대표 경험

### 1. 18 Workspace SaaS API

- Go 기반 대표 포트폴리오 과제
- B2B SaaS API를 로컬에서 완결형으로 재현
- OpenAPI와 smoke/e2e를 함께 정리

### 2. Event / Transaction 트랙

- `15 Event Pipeline`, `17 Game Store Capstone`을 통해 outbox, idempotency, transaction 경계를 반복 학습

## 검증 습관

- 테스트와 smoke를 README 표면에 올립니다.
- 문서와 구현이 drift하지 않도록 OpenAPI와 검증 시나리오를 함께 맞춥니다.

## 성장 방향

Go를 언어 선택으로만 보지 않고, 제품형 API와 운영성을 함께 설계하는 방향으로 깊이를 더하고 싶습니다.
