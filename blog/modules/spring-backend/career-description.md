# Spring Backend 경력기술서 Module

## 한 줄 소개

Spring Boot 기반 서비스에서 인증, JPA 도메인 모델링, Redis/Kafka 연동, Testcontainers 검증까지 함께 정리한 백엔드 개발 경험을 갖고 있습니다.

## 핵심 역량

- Spring Boot / MockMvc / JUnit
- JPA + Flyway
- Redis cart / throttling
- outbox + Kafka notification
- Testcontainers 기반 통합 검증

## 대표 경험

### 1. commerce-backend-v2

- 대표 포트폴리오 capstone
- 같은 도메인을 baseline보다 더 깊게 구현해 면접에서 설명 가능한 결과물로 정리

### 2. Federation / Authorization / Messaging 학습 랩

- 인증 강화, RBAC, 이벤트 경계를 랩 단위로 분리해 학습
- 이후 capstone에서 다시 통합

## 검증 습관

- `make lint`, `make test`, `make smoke`, Compose health 확인을 검증 기준으로 사용합니다.
- Testcontainers 기반 테스트를 통해 런타임 의존성을 코드 수준에서 점검합니다.

## 성장 방향

Spring 생태계를 사용하는 팀에서도 프레임워크 사용을 넘어, 도메인과 운영 기준까지 함께 설계하는 개발자로 성장하고 싶습니다.
