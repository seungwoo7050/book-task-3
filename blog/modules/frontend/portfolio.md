# Frontend Portfolio Module

| 항목 | 내용 |
| --- | --- |
| 포지셔닝 | React, TypeScript, Next.js 기반 제품형 UI 설계와 브라우저 런타임 문제 해결 |
| 대표 프로젝트 | `mini-vrew`, `Ops Triage Console`, `Client Onboarding Portal` |
| 핵심 스택 | React, Next.js, TypeScript, Vite, TanStack Query, React Hook Form, Zod, Playwright |

## 메인 프로젝트

### mini-vrew

브라우저 안에서 업로드, 편집, 자막, 필터, export까지 이어지는 비디오 편집 흐름을 단독으로 설계하고 Vercel에 배포한 프로젝트입니다. FFmpeg WASM, WebGL, Web Audio API, IndexedDB를 실제 UX에 연결한 경험이 프론트엔드 대표 사례가 됩니다.

### front-react 포트폴리오 트랙

`Ops Triage Console`과 `Client Onboarding Portal`을 통해 운영 화면과 고객-facing 플로우를 각각 설계했습니다. optimistic update, retry, validation, draft restore, route guard, E2E 검증을 제품형 UI 관점으로 정리했습니다.

## 마무리

이 모듈은 사용자에게 직접 보이는 경험을 설계하는 역량을 보여 줍니다. 브라우저 API, 상태 전이, 실패 처리, 검증 루프를 함께 보는 프론트엔드 제출본의 핵심 재료입니다.
