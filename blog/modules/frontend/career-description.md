# Frontend 경력기술서 Module

## 한 줄 소개

브라우저 런타임 한계를 고려해 제품형 UI를 설계하고, 검증 자동화까지 함께 정리하는 프론트엔드 개발자입니다.

## 핵심 역량

- 브라우저 API 활용: FFmpeg WASM, WebGL, Web Audio API, IndexedDB
- 제품형 상태 관리: validation, draft restore, optimistic update, retry
- 품질 루프: typecheck, unit test, Playwright E2E

## 대표 경험

### 1. mini-vrew

- 프론트엔드 단독 개발 및 배포
- 브라우저 기반 비디오 편집 경험을 하나의 제품 흐름으로 설계

### 2. front-react 포트폴리오 트랙

- 운영 콘솔과 온보딩 포털을 통해 제품형 UI를 각각 다른 방식으로 구현
- Next.js App Router, React Hook Form, Zod, TanStack Query를 실제 사용자 흐름에 연결

## 검증 습관

- UI도 `typecheck + unit test + E2E`를 함께 묶습니다.
- 화면과 함께 재검증 가능한 명령을 항상 남깁니다.

## 성장 방향

단기적으로는 믿고 맡길 수 있는 프론트엔드 개발자가 되고, 장기적으로는 프론트엔드 품질 기준과 구조를 함께 만드는 개발자로 성장하고 싶습니다.
