# Node Backend 경력기술서 Module

## 한 줄 소개

NestJS 기반 서비스에서 인증, 저장소, 캐시, Swagger, Docker Compose 실행 경로를 함께 정리하는 Node.js 백엔드 개발 경험을 갖고 있습니다.

## 핵심 역량

- NestJS 서비스 구조
- JWT auth / RBAC
- Postgres migration / Redis cache / throttling
- Swagger 문서화와 Docker Compose 재현성

## 대표 경험

### 1. 10 shippable-backend-service

- 채용 제출용 표면으로 다시 패키징한 대표 결과물
- build, test, e2e, run 경로를 README 기준으로 재현 가능하게 구성

### 2. Node Backend Architecture 트랙

- bridge, core, applied 구조로 런타임부터 서비스 표면까지 단계적으로 학습

## 검증 습관

- `pnpm run build`, `pnpm run test`, `pnpm run test:e2e`를 함께 정리합니다.
- 프로젝트의 front door 문서에서 문제, 답, 검증을 바로 찾게 만듭니다.

## 성장 방향

Node.js 환경에서도 프레임워크 기능 나열보다 서비스 표면과 운영성 기준을 함께 설명하는 개발자로 성장하고 싶습니다.
