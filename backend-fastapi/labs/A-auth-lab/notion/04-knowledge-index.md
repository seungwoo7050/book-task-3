# 지식 인덱스

## 재사용 가능한 개념

- 로컬 인증은 회원가입 API 하나가 아니라 credential lifecycle 전체다.
- cookie 인증은 CSRF와 같이 설명해야 설득력이 생긴다.
- 학습용 저장소에서는 첫 실행 성공 경험 자체가 중요한 설계 요구사항이다.

## 용어집

- `refresh token rotation`: refresh token을 재사용하지 않고 매번 교체하는 전략
- `CSRF`: 브라우저가 자동으로 보내는 credential을 악용한 요청을 막기 위한 방어
- `Mailpit`: 로컬 메일 흐름을 눈으로 확인하기 위한 개발용 메일 UI
