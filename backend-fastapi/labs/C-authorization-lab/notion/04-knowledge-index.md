# 지식 인덱스

## 재사용 가능한 개념

- 인증과 인가는 같은 문제가 아니다.
- 권한 규칙은 허용 경로보다 차단 경로 테스트가 더 많은 정보를 준다.
- 이메일 같은 식별자는 초대와 사용자 생성에서 같은 정규화 규칙을 써야 한다.

## 용어집

- `RBAC`: 역할에 따라 권한을 나누는 접근 제어 방식
- `ownership`: 역할과 별개로 "이 리소스를 만든 사람"이라는 추가 경계
- `forbidden-path test`: 허용되면 안 되는 경로를 의도적으로 때려 보는 테스트
