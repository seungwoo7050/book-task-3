# G-ops-lab 학습 노트

이 폴더는 현재 공개용 학습 노트입니다. 운영성 랩의 범위와 한계를 이 노트만으로 이해할 수 있게 정리했습니다.

## 먼저 볼 문서

- 바로 손을 움직여 재현하고 싶다면 [05-development-timeline.md](05-development-timeline.md)부터 읽습니다.
- 왜 이런 구조로 만들었는지부터 이해하고 싶다면 `00`, `01`부터 읽습니다.

## 어떻게 쓰면 좋은가

- 실행 재현이 최우선이면 `05`를 먼저 보고, 운영 질문과 배포 문서의 역할은 `00`, `01`로 보강합니다.
- 운영성 랩이 왜 별도 주제인지 보고 싶다면 `00`, `01`을 먼저 읽습니다.
- health check와 설정 캐시 문제는 `02`에서 빠르게 확인할 수 있습니다.
- 운영 문서와 포트폴리오 설명 감각은 `03`, `04`가 가장 도움이 됩니다.
