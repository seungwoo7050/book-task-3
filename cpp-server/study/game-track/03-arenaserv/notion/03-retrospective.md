# arenaserv 회고

## 이 lab의 가치

`arenaserv`는 “게임 서버 포트폴리오”라는 말을 조금 더 구체적으로 만들어 준다. 중요한 것은 화려한 기능이 아니라, authoritative 상태와 세션 연속성을 설계하고 검증한 흔적을 남기는 일이다.

## 아직 약한 부분

- prediction과 rollback 같은 고급 실시간 기술은 다루지 않는다.
- 운영 배포와 관측성은 별도 주제로 남겨 두었다.

## 포트폴리오로 옮길 때의 조언

- reconnect를 왜 넣었는지 꼭 설명한다.
- 2인뿐 아니라 3인, 4인 검증을 같이 보여 준다.
- ticklab과 연결해 “simulation을 먼저 고정한 뒤 서버로 올렸다”는 학습 흐름을 드러낸다.
