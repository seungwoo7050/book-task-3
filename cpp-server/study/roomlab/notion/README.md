# roomlab 노트 안내

이 폴더는 현재 공개용 학습 노트다. 과거의 긴 디버깅 기록과 타임라인은 [../notion-archive/](../notion-archive/)에 보존했다. 새 `notion/`은 상태 전이와 room lifecycle을 어떻게 읽어야 하는지, 그리고 이를 포트폴리오 설명으로 어떻게 옮기면 좋은지에 초점을 둔다.

## 읽는 순서

2. [01-approach-log.md](01-approach-log.md)
3. [02-debug-log.md](02-debug-log.md)
4. [03-retrospective.md](03-retrospective.md)
5. [04-knowledge-index.md](04-knowledge-index.md)

## 추천 사용법

- 상태 전이가 헷갈리면 00과 04부터 본다.
