# ircserv

flat `study/` 커리큘럼의 pure IRC capstone이다. 앞선 `eventlab`, `msglab`, `roomlab`의 개념을 다시 합쳐 pure TCP C++17 IRC 서버를 만든다.

## Command Surface

- `CAP`
- `PASS`
- `NICK`
- `USER`
- `JOIN`
- `PART`
- `PRIVMSG`
- `NOTICE`
- `TOPIC`
- `MODE`
- `KICK`
- `INVITE`
- `PING`
- `PONG`
- `QUIT`
