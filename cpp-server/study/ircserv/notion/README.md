# ircserv Notion 문서 안내

`ircserv` capstone 프로젝트의 모든 사유와 기록을 이 폴더에서 찾을 수 있습니다. 새로 오셨다면 먼저 아래 순서를 따라 읽어보세요:

1. **00-problem-framing.md** – 세 개의 이전 lab을 어떻게 통합할지에 대한 문제 정의입니다.
2. **01-approach-log.md** – roomlab 기반 확장, advanced IRC 기능 추가 등의 설계 과정을 설명합니다.
3. **02-debug-log.md** – KICK stale state, invite 타이밍 이슈 등 디버깅 히스토리를 제공합니다.
4. **03-retrospective.md** – capstone 의미와 학습 포인트를 되짚어 봅니다.
5. **04-knowledge-index.md** – 통합 과정에서 얻은 네 가지 주요 지식을 정리한 색인입니다.
6. **05-development-timeline.md** – 실제 작성했던 명령어와 테스트 시나리오를 시계열로 나열했습니다.

각 문서는 앞선 lab들과의 관계를 자주 언급하므로, 용도에 맞춰 위 목록에서 골라 읽으면 이해가 쉽습니다.
