# 루트 문서 지도

`docs/`는 이 레포의 상위 설명 계층이다. 개별 lab 문제와 해법은 각 프로젝트 README에서 먼저 보고, 여기서는 왜 이런 트랙 구조를 택했는지와 현재 저장소 상태를 확인한다.

## 먼저 볼 문서

- [curriculum-map.md](curriculum-map.md): 3트랙 구조와 전체 읽기 순서
- [repository-audit.md](repository-audit.md): 현재 저장소가 무엇을 포함하고 무엇을 의도적으로 빼는지
- [path-migration-map.md](path-migration-map.md): 예전 flat 경로와 현재 트랙 경로의 대응표

## 이 디렉터리가 맡는 역할

- 루트 수준에서 커리큘럼 구조를 설명한다.
- 학습 레포로서의 검증 기준과 공개 표면 규칙을 설명한다.
- 구조 개편 전후의 경로 차이를 추적한다.

## 읽을 때 기준

- "문제와 답"은 루트 README와 lab README에서 먼저 본다.
- "왜 이런 순서인가"는 `curriculum-map.md`에서 확인한다.
- "옛 문서와 현재 문서가 어떻게 연결되는가"는 `path-migration-map.md`에서 확인한다.
