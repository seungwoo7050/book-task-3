# 00-language-and-typescript — Notion 문서 가이드

## 이 폴더의 목적

이 `notion/` 폴더는 프로젝트를 Notion으로 옮길 때 그대로 사용할 수 있는 문서 세트다.
단순한 기술 메모가 아니라, **처음 읽는 사람이 프로젝트의 맥락과 흐름을 이해**할 수 있도록 블로그형 에세이 톤으로 작성되어 있다.

## 문서 목록과 읽는 순서

| 순서 | 문서 | 설명 |
|------|------|------|
| 1 | [essay.md](essay.md) | 프로젝트 에세이 — 이 과제를 왜 만들었고, 어떤 판단을 거쳤는지 서사적으로 설명한다 |
| 2 | [timeline.md](timeline.md) | 개발 타임라인 — 소스 코드만으로는 알 수 없는 전체 개발 과정을 순서대로 기록한다 |

## 누가 언제 읽으면 좋은가

- **프로젝트를 처음 접하는 사람**: `essay.md`부터 읽으면 전체 그림이 잡힌다
- **코드를 직접 재현하고 싶은 사람**: `timeline.md`를 따라가면 환경 설정부터 최종 검증까지 순서대로 할 수 있다
- **특정 결정의 근거를 알고 싶은 사람**: `essay.md`에서 해당 주제를 찾으면 왜 그런 선택을 했는지 확인할 수 있다
