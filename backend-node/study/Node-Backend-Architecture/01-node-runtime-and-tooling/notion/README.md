# 01-node-runtime-and-tooling — Notion 문서 가이드

## 이 폴더의 목적

이 `notion/` 폴더는 프로젝트를 Notion으로 옮길 때 그대로 사용할 수 있는 문서 세트다.
처음 읽는 사람이 프로젝트의 맥락과 흐름을 이해할 수 있도록 블로그형 에세이 톤으로 작성되어 있다.

## 문서 목록과 읽는 순서

| 순서 | 문서 | 설명 |
|------|------|------|
| 1 | [essay.md](essay.md) | 프로젝트 에세이 — NDJSON 로그를 다루는 CLI를 왜, 어떻게 만들었는지 서사적으로 설명한다 |
| 2 | [timeline.md](timeline.md) | 개발 타임라인 — 환경 설정부터 최종 검증까지 전체 개발 과정을 순서대로 기록한다 |

## 누가 언제 읽으면 좋은가

- **프로젝트를 처음 접하는 사람**: `essay.md`부터 읽으면 전체 그림이 잡힌다
- **코드를 직접 재현하고 싶은 사람**: `timeline.md`를 따라가면 설치부터 실행까지 빠짐없이 할 수 있다
- **Node.js 스트리밍이나 환경 변수 다루기를 알고 싶은 사람**: `essay.md`의 해당 섹션에서 설계 근거를 확인할 수 있다
