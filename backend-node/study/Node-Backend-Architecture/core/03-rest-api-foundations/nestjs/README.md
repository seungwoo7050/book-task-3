# NestJS 구현

## 담당 범위

module, controller, service, DTO를 통해 framework DI를 체감하는 실무 적용 레인이다.

## 현재 구조

- `src/`: module, controller, service, DTO
- `test/`: unit/e2e 검증

## 실행과 검증

- install: `pnpm install`
- build: `pnpm run build`
- test: `pnpm run test`
- e2e: `pnpm run test:e2e`
