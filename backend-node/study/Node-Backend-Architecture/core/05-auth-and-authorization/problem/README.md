# Problem

## canonical problem statement

JWT 인증과 역할 기반 권한 체크를 Books API에 붙여 요청 권한 경계를 고정하는 것이 canonical problem이다.

## 성공 기준

- 로그인과 보호된 쓰기 경로를 구현할 것
- `401`과 `403`을 테스트에서 구분할 것
- Express와 NestJS 각각의 인증 훅 포인트를 설명할 것

## 제공 자료

- Express/NestJS starter code
- 테스트와 개념 문서
- 장문 학습 로그
