# Problem

## canonical problem statement

프레임워크 없는 Node HTTP 서버로 Books CRUD의 최소 계약을 구현하며 HTTP 기본기를 익히는 것이 문제다.

## 성공 기준

- JSON body parsing과 route dispatch를 직접 구현할 것
- `GET /books`, `GET /books/:id`, `POST /books` 중심 계약을 검증할 것
- status code와 헤더 오류를 테스트로 확인할 것

## 제공 자료

- 문제 설명과 curl 재현 기준
- Node 워크스페이스
- 테스트
- 개념 노트와 학습 로그
