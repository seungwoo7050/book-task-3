# Problem

## canonical problem statement

도서 메타데이터를 다루는 작은 TypeScript CLI를 만들면서 언어 문법, 비동기 흐름, 타입 모델링, 테스트를 한 번에 묶어 익히는 것이 문제다.

## 성공 기준

- 입력 데이터를 타입으로 모델링하고 기본 검증을 수행할 것
- 비동기 유틸리티와 에러 처리를 테스트로 검증할 것
- CLI 실행 예시를 README만 보고 바로 재현할 수 있을 것

## 제공 자료

- 문제 설명과 재현 명령
- TypeScript 워크스페이스
- 테스트 케이스
- 개념 노트와 학습 로그
