# Problem

## canonical problem statement

파일과 환경 변수에 의존하는 작은 CLI를 만들며 Node.js 런타임과 도구 체인을 익히는 것이 문제다.

## 성공 기준

- NDJSON 로그를 줄 단위로 읽고 집계 결과를 출력할 것
- 출력 포맷과 파일 경로 오류를 테스트로 검증할 것
- README의 단일 명령으로 실행 흐름을 다시 재현할 수 있을 것

## 제공 자료

- 샘플 `request-log.ndjson`
- Node 워크스페이스
- 테스트
- 개념 노트와 타임라인
