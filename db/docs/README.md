# Study Docs Index

이 디렉터리는 `study/` 정본 트리의 루트 규칙과 커리큘럼 재설계 근거를 담는다.

## 문서 지도

- [curriculum-audit.md](curriculum-audit.md): 레거시 프로젝트셋의 강점, 약점, 재설계 근거
- [legacy-to-study-map.md](legacy-to-study-map.md): 레거시 모듈과 새 프로젝트의 대응표
- [migration-conventions.md](migration-conventions.md): 디렉터리 모양, 상태 어휘, provenance, 검증 규칙

## 사용 순서

1. 커리큘럼 감사로 왜 새 순서를 택했는지 읽는다.
2. 레거시 매핑으로 어떤 프로젝트가 split, keep, add 되었는지 확인한다.
3. 마이그레이션 규칙으로 각 프로젝트가 어떤 표면을 가져야 하는지 확인한다.
