# Verification vs Search

## CLRS Connection

CLRS Ch 34 separates the cost of verifying a certificate from the cost of discovering one.

## Study Notes

- NP-completeness 학습에서는 solver 구현보다 verifier와 reduction chain이 더 중요한 first step이다.
- vertex cover certificate는 edges를 모두 덮는지 확인하면 된다.
- 3-SAT certificate는 각 clause가 assignment 아래에서 true인지 검사하면 된다.
