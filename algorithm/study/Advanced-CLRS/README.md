# Advanced-CLRS

## 트랙 소개

Core를 지나 CLRS의 고급 주제를 직접 구현 가능한 작은 실험으로 바꿔 보는 심화 트랙이다.

## 왜 이 순서로 배우는가

고급 알고리즘은 읽기만 하면 남지 않는다. 이 트랙은 proof-heavy 챕터를 실행 가능한 과제로 바꿔 학습 장벽을 낮춘다.
