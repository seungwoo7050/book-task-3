        # Problem: Strassen Matrix Multiplication (0x10)

        ## Problem Statement

        Read two n x n integer matrices and print their product. The implementation may internally pad to the next power of two, but the observable output must stay n x n.

        ## Input

        - Line 1: integer n
- Next n lines: matrix A
- Next n lines: matrix B

        ## Output

        Print the product matrix, one row per line.

- `input2.txt`
