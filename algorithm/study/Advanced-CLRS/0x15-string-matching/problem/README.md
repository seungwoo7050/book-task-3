        # Problem: Advanced String Matching (0x15)

        ## Problem Statement

        Given a mode, a text, and a pattern, print all zero-based match positions.

        ## Input

        - Line 1: mode KMP or RK
- Line 2: text
- Line 3: pattern

        ## Output

        Print space-separated zero-based positions, or NONE if no match exists.

        ## Fixtures

        - `input1.txt`
- `input2.txt`

        ## Source

        Repo-authored study problem derived from CLRS Ch 32.
