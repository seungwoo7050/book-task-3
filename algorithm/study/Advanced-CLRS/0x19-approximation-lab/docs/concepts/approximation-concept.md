# Approximation as a Guarantee

## CLRS Connection

CLRS Ch 35 treats approximation algorithms as the compromise between impossible exact optimization and useful deterministic guarantees.

## Study Notes

- set cover의 greedy rule은 harmonic-factor guarantee와 연결된다.
- vertex cover 2-approx는 uncovered edge를 하나 고르고 양 끝점을 모두 채택한다.
- deterministic tie-breaking을 먼저 고정해야 fixture 기반 저장소로 관리하기 쉽다.
