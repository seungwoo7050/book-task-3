        # Problem: Red-Black Tree Insert and Validate (0x12)

        ## Problem Statement

        Process INSERT/FIND/INORDER/VALIDATE commands on a red-black tree with unique integer keys.

        ## Input

        - Line 1: command count q
- Next q lines: commands INSERT x, FIND x, INORDER, VALIDATE

        ## Output

        For each FIND/INORDER/VALIDATE command, print the corresponding result line.

- `input2.txt`
