        # Problem: Number Theory Lab (0x17)

        ## Problem Statement

        Run one of three modes: GCD, CRT, or RSA. Each mode prints a deterministic summary of the computed arithmetic objects.

        ## Input

        - Mode line: GCD, CRT, or RSA
- Remaining lines depend on the mode

        ## Output

        Print the computed summary for the requested mode.

        ## Fixtures

        - `input1.txt`
- `input2.txt`
- `input3.txt`
