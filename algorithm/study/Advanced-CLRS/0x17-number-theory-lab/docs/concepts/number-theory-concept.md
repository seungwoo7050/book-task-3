# From Bezout to RSA

## CLRS Connection

CLRS Ch 31 derives modular inverse from the extended Euclidean algorithm and then uses that inverse in CRT and RSA.

## Study Notes

- extended GCD는 gcd 값뿐 아니라 inverse 계산의 핵심 재료다.
- CRT는 여러 modular constraint를 하나의 congruence로 합친다.
- RSA demo는 small-number toy setting으로만 검증한다.
