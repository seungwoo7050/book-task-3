        # Problem: Computational Geometry Lab (0x16)

        ## Problem Statement

        Run either a HULL query on planar points or an INTERSECT query on two line segments.

        ## Input

        - Mode line: HULL or INTERSECT
- Remaining lines depend on the mode

        ## Output

        For HULL, print hull vertices in CCW order without repeating the first point. For INTERSECT, print YES or NO.

        ## Fixtures

        - `input1.txt`
- `input2.txt`

        ## Source

        Repo-authored study problem derived from CLRS Ch 33.
