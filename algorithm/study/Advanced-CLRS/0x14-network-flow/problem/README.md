        # Problem: Network Flow with Edmonds-Karp (0x14)

        ## Problem Statement

        Given a directed capacitated graph, compute the max flow from source s to sink t.

        ## Input

        - Line 1: n m s t
- Next m lines: u v capacity

        ## Output

        Print the integer max flow value.
