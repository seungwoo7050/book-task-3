# 지식 인덱스 — BOJ 11053에서 건진 개념들

## 핵심 개념

### LIS (Longest Increasing Subsequence)
부분 수열 중 원소가 엄격하게 증가하며 가장 긴 것의 길이.
"부분 수열"은 원래 순서를 유지하되, 원소를 건너뛸 수 있다. "부분 배열(subarray)"과 혼동하지 말 것.

### 1차원 DP 패턴
- 상태 정의: `dp[i]` = 인덱스 i에서 끝나는 LIS 길이
- 초기값: 모든 `dp[i] = 1`
- 전이: `dp[i] = max(dp[j] + 1)` for all `j < i` where `a[j] < a[i]`
- 답: `max(dp)` (마지막이 아닌 전체 최대)

### 엄격 증가 vs 비감소
- 엄격 증가 (`<`): 같은 값 불허 → 이 문제의 요구사항
- 비감소 (`<=`): 같은 값 허용 → 다른 변형 문제에서 사용

## O(N²) vs O(N log N)

| 방식 | 아이디어 | 적용 범위 |
|------|---------|-----------|
| DP 이중 루프 | 모든 이전 위치 탐색 | $N \le 10^3$ ~ $10^4$ |
| Patience sorting + bisect | tails 배열 유지, 이분탐색 | $N \le 10^5$ ~ $10^6$ |

이 문제에서는 $N \le 1\,000$이므로 DP 이중 루프가 적합했다.
