# 트리의 지름 개념 정리

## CLRS 연결
트리의 지름은 CLRS에 직접 없지만, BFS 기반 증명은 CLRS Ch 22.2의 BFS 성질을 활용.

## 핵심 아이디어 (Two-BFS)
1. 임의의 노드에서 BFS → 가장 먼 노드 $u$
2. $u$에서 BFS → 가장 먼 노드 $v$
3. $u$-$v$ 거리가 트리의 지름

## 정당성 증명
BFS로 가장 먼 노드를 찾으면 그 노드는 반드시 지름의 한 끝점이다.
(귀류법: 끝점이 아니라면 더 먼 노드가 존재하여 모순)

## 복잡도
BFS 2회 = $O(V + E) = O(V)$ (트리이므로 $E = V-1$).
