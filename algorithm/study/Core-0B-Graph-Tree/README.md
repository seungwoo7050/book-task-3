# Core-0B-Graph-Tree

## 트랙 소개

트리 구조를 별도 자료형으로 다루며, 부모 찾기, 순회, 지름 계산 같은 대표 패턴을 익히는 트랙이다.

## 왜 이 순서로 배우는가
