# BOJ 12865 — 회고

## 배운 것

0/1 배낭의 1D 최적화에서 **역순 순회의 정당성**을 완전히 이해한 것이 가장 큰 수확이다. "현재 행이 이전 행에만 의존"하는 구조에서, 역순 > 순회하면 "아직 갱신되지 않은 값 = 이전 행의 값"을 보장한다.

## 순방향은 완전 배낭

순방향으로 순회하면 각 아이템을 여러 번 사용하는 **완전 배낭**(unbounded knapsack)이 된다. 역순이 0/1 배낭, 순방향이 완전 배낭 — 이 차이를 코드 한 줄(`range` 방향)로 제어할 수 있다는 것이 흥미롭다.

## DP 차원 축소의 일반 원리

1149에서는 이전 행 3개 변수, 2748에서는 이전 2개 변수, 이 문제에서는 1D 배열 역순. 모두 같은 원리: "새 상태가 이전 상태에만 의존하면, 차원을 줄일 수 있다."
