# Problem: Normal Knapsack (BOJ 12865)

## Problem Statement

Given $N$ items, each with weight $W_i$ and value $V_i$, and a knapsack with capacity $K$, find the maximum total value of items that fit in the knapsack. Each item can be used at most once (0/1 knapsack).

## Input

- Line 1: $N$, $K$ ($1 \le N \le 100$, $1 \le K \le 100\,000$)
- Next $N$ lines: $W_i$, $V_i$ ($1 \le W_i \le 100\,000$, $0 \le V_i \le 1000$)

## Output

Print the maximum total value.

## Examples

**Input**:
```
4 7
6 13
4 8
3 6
5 12
```

**Output**: `14`
