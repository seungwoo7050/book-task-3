# BOJ 1149 — RGB 거리: 상태 DP

## 첫인상

$N$채의 집을 빨강/초록/파랑으로 칠하되, 인접한 집과 다른 색이어야 한다. 비용을 최소화하는 문제. 2748의 1차원 DP에서 "상태"라는 차원이 추가된다 — 각 집에서 "어떤 색으로 끝나느냐"가 상태.

## 문제 구조

- $N$채의 집 ($N \leq 1000$)
- 각 집에 3가지 색의 비용이 주어짐
- 인접한 집은 다른 색이어야 함
- 최소 총 비용 출력

## 왜 이 문제를 선택했는가

CLRS Ch 15의 DP에서 "상태 정의"가 핵심인 문제. 상태를 `(집 번호, 마지막 색상)`으로 정의하면 점화식이 자연스럽게 도출된다.

## 난이도 평가

Silver 등급. 상태 정의만 올바르면 구현은 직관적. 공간 최적화(이전 행만 유지)까지 자연스럽게 가능.
