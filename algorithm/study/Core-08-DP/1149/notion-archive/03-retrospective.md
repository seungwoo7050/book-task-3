# BOJ 1149 — 회고

## 배운 것

DP의 상태 정의가 핵심이다. "어떤 결정을 내렸느냐"가 상태에 포함되어야 한다. 피보나치는 상태가 인덱스 하나였지만, RGB 거리는 "(인덱스, 색상)" 쌍이다. 이 패턴은 배낭 문제의 "(아이템, 남은 용량)" 상태와 같은 구조.

## 공간 최적화의 일반 원리

$\textrm{dp}[i]$가 $\textrm{dp}[i-1]$에만 의존하면, 이전 행만 유지하면 된다. 피보나치에서는 변수 2개, 여기서는 변수 3개(리스트 1개). 12865(배낭)에서는 1D 배열의 역순 순회로 같은 원리를 적용한다.
