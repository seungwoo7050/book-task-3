# Core-08-DP

## 트랙 소개

점화식, 상태 정의, 전이 방향을 가장 기본적인 형태부터 차근차근 고정하는 트랙이다.

## 왜 이 순서로 배우는가

DP는 풀이를 외우면 금방 무너지고, 상태 정의를 스스로 세워야 오래 간다. 그래서 작은 1차원 DP부터 배낭 문제까지 묶는다.
