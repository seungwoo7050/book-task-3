# DP 개념 정리 — 피보나치 수 2

## CLRS 연결
CLRS Ch 15.1 Rod Cutting — DP의 핵심인 **중복 부분 문제**와 **메모이제이션**.
피보나치는 DP의 가장 기본적인 예시.

## 점화식
$$F(n) = F(n-1) + F(n-2), \quad F(0)=0,\; F(1)=1$$

## 구현 방식
| 방식 | 시간 | 공간 | 설명 |
|------|------|------|------|
| 재귀 (naive) | $O(2^n)$ | $O(n)$ | 지수적 폭발 |
| 메모이제이션 | $O(n)$ | $O(n)$ | Top-down |
| 반복 (bottom-up) | $O(n)$ | $O(1)$ | **최적** |

## 채택: Bottom-up 공간 최적화
`a, b = b, a + b` — 두 변수만으로 $O(1)$ 공간.
$n \le 90$이므로 64비트 정수 범위를 넘지만 Python은 BigInt 지원.
