# Problem: Fibonacci Number 2 (BOJ 2748)

## Problem Statement

Given $n$ ($0 \le n \le 90$), compute the $n$-th Fibonacci number.

$F(0) = 0$, $F(1) = 1$, $F(n) = F(n-1) + F(n-2)$ for $n \ge 2$.

## Input

One integer $n$.

## Output

Print $F(n)$.

## Examples

**Input**: `10` → **Output**: `55`
