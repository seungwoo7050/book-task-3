# Problem: Sort Numbers (BOJ 2750)

## Problem Statement

Given $N$ integers, sort them in ascending order.

## Input

- Line 1: integer $N$ ($1 \le N \le 1000$)
- Next $N$ lines: one integer each (absolute value ≤ 1 000 000), all distinct.

## Output

Print the sorted numbers, one per line.

## Examples

**Input**:
```
5
5
2
3
4
1
```

**Output**:
```
1
2
3
4
5
```
