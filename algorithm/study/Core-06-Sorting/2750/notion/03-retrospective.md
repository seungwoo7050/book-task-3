# BOJ 2750 — 회고

## 배운 것

Python의 내장 `sort()`는 Timsort 알고리즘으로, 최악 $O(N \log N)$, 부분 정렬된 데이터에서 $O(N)$까지 내려간다. 실전에서는 이 내장 정렬이 거의 항상 최선의 선택이다.

## 정렬의 하한

비교 기반 정렬의 하한은 $\Omega(N \log N)$ (CLRS Ch 8.1 결정 트리 논증). 이보다 빠른 정렬(계수/기수/버킷)은 입력에 대한 가정이 필요하다.
