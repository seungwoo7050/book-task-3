# BOJ 1181 — 개념 인덱스

## 핵심 개념

| 개념 | CLRS 참조 | 설명 |
|------|-----------|------|
| 다중 키 정렬 | Ch 8 | 1차: 길이, 2차: 사전순 |
| 안정 정렬 | Ch 2, 8 | 동일 키에서 원래 순서 유지 |

## 시간 복잡도

- set 구성: $O(N \cdot L)$ — $L$은 평균 단어 길이
- 정렬: $O(N \log N \cdot L)$ — 문자열 비교 비용 포함

## 연결 문제

- **BOJ 2750** (Core-06): 기본 정렬
- **BOJ 2170** (Core-06): 정렬 후 스위프
- **BOJ 11650**: 좌표 정렬 — 다중 키 정렬 변형
