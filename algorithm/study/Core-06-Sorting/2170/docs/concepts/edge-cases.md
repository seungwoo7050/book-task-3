# 경계 사례 점검 — BOJ 2170 Drawing Lines

## 1. 선분이 1개
단순히 `y - x` 출력.

## 2. 모든 선분이 겹침
`[1,10], [2,5], [3,7]` → 10 - 1 = 9.

## 3. 완전히 분리된 선분
`[1,2], [5,6]` → 1 + 1 = 2.

## 4. 포함 관계
`[1,10], [3,5]` → 뒤 선분이 앞에 완전 포함.

## 5. 좌표 범위 $10^9$
`int` 범위 주의. Python은 자동 처리.

## 핵심 주의점
- `sys.stdin.readline` 필수 ($N \le 10^6$)
- 마지막 구간 처리 잊지 말 것
