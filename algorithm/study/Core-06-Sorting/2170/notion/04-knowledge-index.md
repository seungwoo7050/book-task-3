# BOJ 2170 — 개념 인덱스

## 핵심 알고리즘

| 개념 | CLRS 참조 | 설명 |
|------|-----------|------|
| 정렬 후 스위프 | Ch 33 (기하) | 정렬을 전처리로 활용 |
| 구간 합치기 (Interval Merging) | — | 겹치는 구간을 선형 탐색으로 합침 |

## 시간 복잡도

- 정렬: $O(N \log N)$
- 스위프: $O(N)$
- 전체: $O(N \log N)$

## 연결 문제

- **BOJ 1931** (Core-09): 회의실 배정 — 정렬 후 그리디
- **BOJ 2750** (Core-06): 기본 정렬
- **BOJ 1181** (Core-06): 다중 키 정렬
- **BOJ 0x16 계산 기하** (Advanced-CLRS): 기하 스위프 알고리즘
