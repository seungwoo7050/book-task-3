# 활동 선택 문제 개념 정리 — 회의실 배정

## CLRS 연결
CLRS Ch 16.1 Activity-Selection Problem — 그리디 알고리즘의 대표 예시.

## 핵심 아이디어
**종료 시간 기준 정렬** 후, 현재 종료 시간 이후에 시작하는 회의를 탐욕적으로 선택.

## 정당성 증명 (Exchange Argument)
최적해에서 가장 먼저 끝나지 않는 회의를 가장 먼저 끝나는 회의로 교환해도
나머지 회의에 영향 없이 ≥ 같은 수의 회의를 선택할 수 있다.

## 정렬 키
`(end, start)` — 종료 동일 시 시작이 빠른 것 우선.
`start == end` (시작 = 종료)인 회의도 가능하므로 이 정렬이 중요.
