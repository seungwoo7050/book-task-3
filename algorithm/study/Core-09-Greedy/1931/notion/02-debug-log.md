# BOJ 1931 — 디버깅 기록

## 함정 1: 정렬 키 실수

**증상**: 종료 시간만으로 정렬하면 같은 종료 시간에서 순서가 불확정

**예시**: (2,2)와 (1,2) → (2,2)가 먼저 오면, last_end=2 후 (1,2)는 start=1 < 2로 탈락

**해결**: `(end, start)` 복합 키 정렬

## 함정 2: 경계 조건

**증상**: `start > last_end` vs `start >= last_end`

**원인**: 문제에서 "종료 시각에 바로 시작 가능"이라 명시

**해결**: `start >= last_end` 사용
