# 수 묶기 그리디 개념 정리

## 핵심 아이디어
수를 2개씩 묶어 곱으로 합산할 수 있을 때 최대 합을 구하는 문제.
양수, 음수, 0, 1을 분리하여 각각 최적 전략을 적용.

## 전략
| 그룹 | 전략 |
|------|------|
| 양수 (>1) | 큰 것끼리 묶어 곱 |
| 1 | 묶지 않고 더함 (1×x < 1+x) |
| 음수 (<0) | 작은 것끼리 묶어 양수화 |
| 0 | 홀수 남은 음수와 묶어 상쇄 |

## CLRS 연결
CLRS Ch 16 Greedy — 최적 부분 구조.
각 그룹 내에서 독립적으로 최적 선택 가능.
