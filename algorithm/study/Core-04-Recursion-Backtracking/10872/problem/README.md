# Problem: Factorial (BOJ 10872)

## Problem Statement

Given a non-negative integer $N$ ($0 \le N \le 12$), compute $N!$.

## Input

One integer $N$.

## Output

Print $N!$.

## Examples

**Input**: `10` → **Output**: `3628800`  
**Input**: `0` → **Output**: `1`
