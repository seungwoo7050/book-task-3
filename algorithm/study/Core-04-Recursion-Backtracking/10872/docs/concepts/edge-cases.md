# Edge Cases — BOJ 10872 Factorial

## 1. N = 0

```
입력: 0
출력: 1
```
$0! = 1$은 수학적 정의. 기저 조건에서 바로 반환.

## 2. N = 1

```
입력: 1
출력: 1
```
$1! = 1$. `n <= 1` 조건으로 0!과 함께 처리.

## 3. N = 12 (최대)

```
입력: 12
출력: 479001600
```
$12! = 479{,}001{,}600$. 32비트 정수 범위 내 (최대 ~$2.1 \times 10^9$).

## 4. 오버플로 고려

$13! = 6{,}227{,}020{,}800 > 2^{32}$이므로 C++에서 `long long` 필요.
이 문제는 $N \le 12$이므로 `int`로도 충분하지만, 안전을 위해 `long long` 사용.
Python은 임의 정밀도이므로 무관.

## 핵심 주의점

- **$0! = 1$**: 반드시 처리해야 하는 기저 조건
- 기저 조건을 `n == 0`이 아닌 `n <= 1`로 하면 0과 1 모두 처리 가능
- 재귀 깊이 최대 12 → Python 기본 한도(1000) 내
