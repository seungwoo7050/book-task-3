# BOJ 9663 — 개념 인덱스

## 핵심 알고리즘

| 개념 | CLRS 참조 | 설명 |
|------|-----------|------|
| 백트래킹 | Ch 4 | 탐색 트리에서 유망하지 않은 가지 제거 |
| 가지치기 (Pruning) | — | 충돌 발생 시 해당 서브트리 전체 건너뜀 |
| N-Queen | — | 백트래킹의 대표 응용 문제 |

## 자료구조 사용

- **col 배열**: 열 사용 여부 $O(N)$
- **diag1 배열**: 좌상→우하 대각선 $O(2N)$
- **diag2 배열**: 우상→좌하 대각선 $O(2N)$

## 시간 복잡도

- 최악: $O(N!)$ — 가지치기 후 실제 탐색 노드
- $N = 15$일 때 약 2,279,184개 (정답)
