# Problem: N-Queen (BOJ 9663)

Place $N$ queens on an $N \times N$ chessboard so that no two queens attack each other (same row, column, or diagonal). Count the number of valid placements.

## Input
$N$ ($1 \le N \le 14$)

## Output
Number of solutions.

## Example
**Input**: `8` → **Output**: `92`
