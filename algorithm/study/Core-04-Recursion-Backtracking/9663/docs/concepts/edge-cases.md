# Edge Cases — BOJ 9663 N-Queen

## 1. N = 1
→ `1`. 1×1 보드에 퀸 하나.

## 2. N = 2, 3
→ `0`. 해가 존재하지 않음.

## 3. N = 4
→ `2`. 최소 해가 존재하는 경우.

## 4. N = 14 (최대)
→ `365596`. Python으로는 시간 초과 위험. C++ 사용 권장.

## 핵심 주의점
- 대각선 인덱싱: `row - col`이 음수일 수 있으므로 `+ N - 1` 오프셋
- Python 성능: N=14에서 ~10초+ 소요 가능. PyPy 또는 C++ 필요
- 상태 복원: 세 배열 모두 `False`로 복원 필수
