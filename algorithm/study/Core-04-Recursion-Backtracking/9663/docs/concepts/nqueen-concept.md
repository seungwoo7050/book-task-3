# N-Queen 개념 정리

## 문제 정의

$N \times N$ 체스판에 $N$개의 퀸을 배치하되, 어떤 두 퀸도 같은 행/열/대각선에 있지 않도록 한다.

## 백트래킹 접근

### 행 단위 배치
행 0부터 행 $N-1$까지 순서대로 퀸을 배치.
각 행에 정확히 하나의 퀸 → 행 충돌 자동 방지.

### $O(1)$ 충돌 검사
세 개의 boolean 배열:
- `col[c]`: 열 $c$ 사용 여부
- `diag1[row - col + N - 1]`: `\` 대각선
- `diag2[row + col]`: `/` 대각선

### 대각선 인덱싱
- `\` 대각선: `row - col`이 같은 칸들. 음수 방지를 위해 `+ N - 1`
- `/` 대각선: `row + col`이 같은 칸들

## 해의 개수

| N | 해의 수 |
|---|---------|
| 1 | 1 |
| 4 | 2 |
| 8 | 92 |
| 12 | 14200 |
| 14 | 365596 |

## 가지치기 효과

$N = 8$일 때:
- 완전 탐색: $8^8 = 16{,}777{,}216$
- 순열로 줄이면: $8! = 40{,}320$
- 대각선 가지치기: 실제 탐색 ~2000 노드

백트래킹으로 탐색 공간이 극적으로 줄어든다.
