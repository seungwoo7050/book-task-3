# Problem: N and M (1) — BOJ 15649

Print all sequences of length $M$ chosen from $\{1, 2, \dots, N\}$ without repetition (permutations), in lexicographic order.

## Input
$N$, $M$ ($1 \le M \le N \le 8$)

## Output
One sequence per line, space-separated.

## Example
**Input**: `3 1`  
**Output**:
```
1
2
3
```
