# 경계 사례 점검 — BOJ 15649 N and M (1)

## 1. M = 1
각 숫자가 한 줄에 하나씩 출력. 총 $N$개.

## 2. M = N
$N!$개의 완전 순열 출력.

## 3. N = M = 1
```
입력: 1 1
출력: 1
```

## 4. N = 8, M = 8 (최대)
$8! = 40{,}320$줄 출력. 출력 버퍼링 필수.

## 핵심 주의점
- 사전순 출력 → 1부터 N까지 오름차순 시도로 자동 보장
- 출력 최적화: 개별 print() 대신 리스트에 모아서 join
- 상태 복원: `seq.pop()` + `used[i] = False` 쌍 필수
