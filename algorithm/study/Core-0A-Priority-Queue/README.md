## 왜 이 순서로 배우는가

우선순위 큐는 알고리즘 곳곳에서 재사용된다. 여기서 최소/최대 힙과 힙 기반 그리디를 묶어 두면 이후 그래프 문제에서 다시 쓰기 쉽다.
