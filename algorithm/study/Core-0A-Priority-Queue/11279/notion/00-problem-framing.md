# BOJ 11279 — 최대 힙

## 첫인상

CLRS Ch 6 힙의 기본 연산을 구현하는 문제. 삽입(Insert)과 최댓값 추출(Extract-Max) 두 연산을 지원하는 최대 힙. Python의 `heapq`는 최소 힙이므로, 부호를 뒤집어 최대 힙으로 사용하는 트릭이 핵심.

## 문제 구조

- $N$개의 연산
- $x > 0$: 삽입
- $x = 0$: 최댓값 추출 (비어있으면 0 출력)

## 왜 이 문제를 선택했는가

힙의 기초. `heapq` 라이브러리 사용법과 최소 힙 ↔ 최대 힙 변환 트릭을 익히기 위함.

## 난이도 평가

Bronze. 라이브러리만 알면 직관적.
