# Core-05-Simulation

## 트랙 소개

문제 설명을 상태 전이 규칙으로 번역하는 힘을 키우는 트랙이다. 구현은 길어도 논리는 짧게 정리하는 연습이 핵심이다.

## 왜 이 순서로 배우는가

시뮬레이션은 실수하기 쉬운 대신, 문서화가 잘되면 실력을 보여 주기 좋은 분야다. 상태 표와 규칙 분리를 익히기 좋다.
