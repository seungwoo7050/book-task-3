# Problem: Scale (BOJ 2920)

## Problem Statement

Given a sequence of 8 integers (a permutation of 1 through 8), determine whether the sequence is:

- **ascending** if it is `1 2 3 4 5 6 7 8`
- **descending** if it is `8 7 6 5 4 3 2 1`
- **mixed** otherwise

## Input

One line containing 8 integers separated by spaces. Each integer is between 1 and 8, and all are distinct.

## Output

Print `ascending`, `descending`, or `mixed`.

## Examples

**Input**: `1 2 3 4 5 6 7 8` → **Output**: `ascending`  
**Input**: `8 7 6 5 4 3 2 1` → **Output**: `descending`  
**Input**: `8 1 7 2 6 3 5 4` → **Output**: `mixed`
