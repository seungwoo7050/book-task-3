# BOJ 14503 — 로봇 청소기: 방향 전환 시뮬레이션

## 첫인상

격자 위에서 로봇이 정해진 규칙대로 움직이며 청소하는 문제. 시뮬레이션 문제 중에서 "방향"이 핵심 상태인 유형이다. 현재 바라보는 방향에 따라 왼쪽 회전, 전진, 후진이 결정되고, 이를 정확히 구현하면 된다.

## 문제 구조 파악

- $N \times M$ 격자, 0은 빈 칸, 1은 벽
- 로봇의 초기 위치 $(r, c)$와 방향 $d$ (0=북, 1=동, 2=남, 3=서)
- 동작 규칙:
  1. 현재 칸 청소
  2. 왼쪽으로 회전하며 4방향 탐색, 청소하지 않은 빈 칸이 있으면 전진
  3. 4방향 모두 청소 완료/벽이면 후진 (방향 유지). 뒤가 벽이면 정지

## 왜 이 문제를 선택했는가

Core-05의 Gold 문제로, 방향 관리 + 격자 탐색 + 상태 기계(state machine)의 조합을 훈련한다. 글로벌 네 방향(dr, dc)과 회전/후진 연산의 정확한 구현이 핵심.

## 난이도 평가

Gold 등급. 알고리즘 자체보다 복잡한 규칙을 빠짐없이 구현하는 **구현력**이 핵심. 규칙을 하나라도 잘못 해석하면 답이 달라진다.
