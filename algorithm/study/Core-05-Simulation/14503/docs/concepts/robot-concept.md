# 로봇 청소기 시뮬레이션 개념 정리

## 핵심 아이디어
상태 기반 시뮬레이션. 로봇의 현재 위치와 방향이 상태.
규칙에 따라 상태를 전이하며 청소한 칸 수를 카운트.

## 동작 규칙
1. 현재 칸 청소
2. 반시계방향으로 4방향 탐색:
   - 청소 안 된 빈 칸 발견 → 이동 + 방향 전환
3. 4방향 모두 막혀있으면:
   - 뒤로 이동 가능 → 후진 (방향 유지)
   - 뒤가 벽 → 정지

## 방향 회전
- 좌회전: `(d + 3) % 4`
- 후진 방향: `(d + 2) % 4`

while True:
