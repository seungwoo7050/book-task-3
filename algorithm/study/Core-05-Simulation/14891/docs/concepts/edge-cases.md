# 경계 사례 점검 — BOJ 14891 Gear

## 1. 모든 톱니가 같은 극
전파 없음. 지정된 톱니바퀴만 회전.

## 2. 모든 접촉점이 다른 극
전체 4개 톱니바퀴가 번갈아 회전.

## 3. K = 1
한 번의 회전만 수행.

## 4. 연속 회전으로 상태 변화
이전 회전의 결과가 다음 회전의 전파 조건에 영향.

## 핵심 주의점
- 전파를 먼저 결정한 뒤 동시 회전 (순서 의존성 제거)
- deque 회전 방향: 시계=오른쪽→왼쪽, 반시계=왼쪽→오른쪽
- 톱니바퀴 번호 1-indexed → 0-indexed 변환
