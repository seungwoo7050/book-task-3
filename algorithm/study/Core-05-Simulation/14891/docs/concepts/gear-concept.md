# 톱니바퀴 시뮬레이션 개념 정리

## 핵심 아이디어
4개의 톱니바퀴가 맞물려 있으며, 하나를 회전하면 인접 톱니바퀴가
접촉점의 극성이 다를 때 반대 방향으로 연쇄 회전한다.

## 구현 패턴
1. 회전 전에 모든 전파 방향을 먼저 결정 (left, right propagation)
2. 결정 후 동시에 회전 적용

## 자료구조
`deque`로 원형 배열을 표현:
- 시계방향: `appendleft(pop())` — 오른쪽 끝을 왼쪽으로
- 반시계방향: `append(popleft())` — 왼쪽 끝을 오른쪽으로

## 접촉점
- 오른쪽 접촉점: index 2
- 왼쪽 접촉점: index 6

극성이 다르면 반대 방향으로 회전, 같으면 전파 중단.
