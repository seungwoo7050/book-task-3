# BOJ 14891 — 개념 인덱스

## 핵심 개념

| 개념 | 설명 |
|------|------|
| 시뮬레이션 | 규칙 기반 상태 전이 |
| 연쇄 반응 전파 | 조건 만족 시 인접 객체에 영향 전달 |
| 원형 배열 회전 | deque로 시계/반시계 회전 구현 |

## 자료구조 사용

- **deque**: 톱니바퀴의 원형 구조 표현 + 회전 연산 $O(1)$
- **dirs 배열**: 전파 결과 저장 (판단과 적용 분리)

## 시간 복잡도

- 각 명령당 $O(1)$ (톱니바퀴 4개, 기어 8개 — 모두 상수)
- 전체: $O(K)$

## 연결 문제

- **BOJ 14503** (Core-05): 로봇 청소기 — 방향 + 이동 시뮬레이션
- **BOJ 15662**: 톱니바퀴 2 — 이 문제의 확장 (톱니바퀴 T개)
- **BOJ 5430** (Core-02): AC — deque 활용
