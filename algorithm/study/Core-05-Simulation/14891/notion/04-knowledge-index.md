# 지식 인덱스

> 프로젝트: 톱니바퀴
> 아래 내용은 `notion-archive/04-knowledge-index.md`에 남아 있던 기존 학습 기록을 현재 공개 노트 기준으로 다듬은 버전이다.

## 핵심 개념

| 개념 | 설명 |
|------|------|
| 시뮬레이션 | 규칙 기반 상태 전이 |
| 연쇄 반응 전파 | 조건 만족 시 인접 객체에 영향 전달 |
| 원형 배열 회전 | deque로 시계/반시계 회전 구현 |

## 자료구조 사용

- **deque**: 톱니바퀴의 원형 구조 표현 + 회전 연산 $O(1)$
- **dirs 배열**: 전파 결과 저장 (판단과 적용 분리)

## 시간 복잡도

- 각 명령당 $O(1)$ (톱니바퀴 4개, 기어 8개 — 모두 상수)
- 전체: $O(K)$
