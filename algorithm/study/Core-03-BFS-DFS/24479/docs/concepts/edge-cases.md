# 경계 사례 점검 — BOJ 24479 DFS 방문 순서

## 1. 고립된 정점 (간선 없음)

```
입력: N=3, M=0, R=1
출력: 1, 0, 0
```
시작점만 방문 순서 1, 나머지는 0.

## 2. 일자형 그래프 (체인)

```
1-2-3-4-5, R=1
출력: 1, 2, 3, 4, 5
```
한 방향으로만 탐색.

## 3. 완전 그래프

모든 정점이 연결. 오름차순 정렬에 의해 항상 작은 번호부터 방문.

## 4. 연결 요소가 여러 개

시작점과 같은 컴포넌트만 방문. 다른 컴포넌트의 정점은 0.

## 5. N = 1

```
입력: N=1, M=0, R=1
출력: 1
```
정점 하나만 존재.

## 6. 최대 입력 (N=100,000, M=200,000)

- Python: `setrecursionlimit(200000)` 필수
- 인접 리스트 정렬: $O(M \log M)$이 병목

## 7. 시작점이 1이 아닌 경우

```
N=5, M=4, R=3
```
3번 정점부터 시작. 방문 순서가 1번부터가 아님에 주의.

## 핵심 주의점

- **인접 리스트 정렬**: 오름차순 정렬 안 하면 오답
- **재귀 한도**: Python에서 `setrecursionlimit` 필수
- **0-indexed vs 1-indexed**: 정점 번호 1부터 시작
- **미방문 정점**: 0 출력 (방문 순서가 아닌 특수값)
