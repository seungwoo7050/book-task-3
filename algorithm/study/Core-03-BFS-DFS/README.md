# Core-03-BFS-DFS

## 트랙 소개

정점 방문 순서, 그래프 표현, 격자 탐색을 묶어 BFS/DFS의 기본기를 고정하는 트랙이다.

## 왜 이 순서로 배우는가

그래프 탐색은 이후 트리, 최단 경로, 위상 정렬의 기반이다. 방문 처리와 큐/스택 규칙을 여기서 확실히 잡아야 뒤가 편해진다.
