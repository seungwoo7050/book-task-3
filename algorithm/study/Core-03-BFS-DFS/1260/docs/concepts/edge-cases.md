# Edge Cases — BOJ 1260 DFS and BFS

## 1. 정점 하나 (N=1)

```
입력: 1 0 1
출력:
1
1
```
간선 없이 시작점만 출력.

## 2. 완전 그래프

모든 정점이 연결. 오름차순 방문 시 DFS와 BFS 모두 1→2→3→...→N.

## 3. 일자형 그래프 (체인)

```
1-2-3-4-5, V=1
DFS: 1 2 3 4 5
BFS: 1 2 3 4 5
```
같은 결과이지만 탐색 방식이 다름.

## 4. 별 모양 그래프

```
1-2, 1-3, 1-4, V=1
DFS: 1 2 3 4
BFS: 1 2 3 4
```
중심에서 모든 정점이 직접 연결.

## 5. 연결되지 않은 정점

시작점과 같은 컴포넌트만 출력.
```
3 1 1
1 2
DFS: 1 2
BFS: 1 2
```
정점 3은 출력에 포함되지 않음.

## 6. 중복 간선

문제에서 중복 간선이 주어질 수 있음.
인접 리스트에 같은 정점이 여러 번 들어가도 `visited` 체크로 무시됨.

## 핵심 주의점

- **BFS visited 타이밍**: 큐에 넣을 때 체크 (꺼낼 때 X)
- **DFS visited 타이밍**: 방문 시 즉시 체크
- 인접 리스트 정렬 필수 (작은 번호 우선)
- 두 탐색에서 `visited` 배열을 각각 독립적으로 사용
