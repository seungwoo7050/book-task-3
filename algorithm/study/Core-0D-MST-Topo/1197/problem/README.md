# Problem: 최소 스패닝 트리 (BOJ 1197)

## Problem Statement

무방향 가중 그래프가 주어졌을 때, 최소 스패닝 트리의 가중치 합을 구하라.

## Input

- Line 1: $V$, $E$ ($1 \le V \le 10\,000$, $1 \le E \le 100\,000$)
- Next $E$ lines: $A$, $B$, $C$ (간선, 가중치 $|C| \le 1\,000\,000$)

## Output

MST의 가중치 합.

## Examples

**Input**:
```
3 3
1 2 1
2 3 2
1 3 3
```

**Output**:
```
3
```
