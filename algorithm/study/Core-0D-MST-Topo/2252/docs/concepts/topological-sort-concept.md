# 위상 정렬 개념 정리 — 줄 세우기

## CLRS 연결
CLRS Ch 22.4 Topological Sort — DAG에서 선후 관계를 선형 순서로 나열.

## 핵심 아이디어
**Kahn's Algorithm** (BFS 기반):
1. 진입 차수(in-degree)가 0인 노드를 큐에 삽입
2. 큐에서 꺼내 결과에 추가, 인접 노드의 진입 차수 감소
3. 진입 차수가 0이 되면 큐에 삽입
4. 모든 노드 처리 시 완료

## DFS 기반 방법
CLRS 원본은 DFS 종료 시각 역순. 구현이 더 복잡.

## 복잡도
$O(V + E)$.
