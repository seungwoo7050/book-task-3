## 읽는 법

1. 원하는 프로젝트의 `00-series-map.md`를 먼저 연다.
2. `10-development-timeline.md`에서 문제 이해, 가설, 첫 구현 과정을 따라간다.
3. `20-development-timeline.md`에서 검증, edge case, 최종 정리를 확인한다.
