# Bronze/Silver/Gold 탐색형 학습 로그 인덱스

이 디렉터리는 `algorithm/study`의 Core 트랙 42개 독립 프로젝트를 다시 읽기 좋게 묶은 블로그 입구다. 각 시리즈는 문제 계약, 첫 구현, fixture 검증, 마지막 개념 정리 순서로 흐르기 때문에 한 문제를 처음부터 끝까지 따라가기 쉽다.

기존 초안은 `study/blog/_legacy/2026-03-13-isolate-and-rewrite`에 따로 보관했다. 지금 보이는 본문은 `source/README/docs/tests/CLI`를 기준으로 다시 다듬은 버전이다.

## 읽는 법

1. 관심 있는 프로젝트의 `00-series-map.md`에서 질문과 읽는 순서를 먼저 잡는다.
2. `01-evidence-ledger.md`에서 어떤 근거로 chronology를 복원했는지 확인한다.
3. `10-development-timeline.md`와 `20-development-timeline.md`를 이어 읽으며 구현과 검증이 어디서 갈렸는지 따라간다.
