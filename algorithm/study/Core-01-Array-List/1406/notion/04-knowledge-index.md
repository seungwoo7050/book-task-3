# 지식 인덱스 — BOJ 1406

## 핵심 개념

### Two-Stack 모델
커서 왼쪽과 오른쪽을 각각 스택으로 관리하여, 커서 위치에서의 삽입/삭제/이동을 O(1)로 수행하는 기법.
연결 리스트의 대안으로 Python에서 특히 유효하다.

### 배열 insert/pop(i)의 비용
Python list의 `insert(i, x)`와 `pop(i)`는 O(n). 끝에서의 `append()`와 `pop()`만 amortized O(1).
