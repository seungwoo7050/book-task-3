# 지식 인덱스 — BOJ 10807

## 핵심 개념

### 선형 탐색 (Linear Search)
배열을 처음부터 끝까지 순회하면서 조건에 맞는 원소를 찾거나 세는 O(n) 연산.
가장 단순하지만 정렬되지 않은 데이터에서는 이것이 최선이다.

### Python list.count()
내장 메서드로, C 레벨에서 선형 탐색을 수행한다.
직접 루프를 짜는 것보다 빠르지만 복잡도는 동일하게 O(n).
