# 경계 사례 점검 — BOJ 2164 Card2

## 1. N = 1

```
입력: 1
출력: 1
```
카드가 하나뿐이면 아무 동작 없이 바로 출력. `while len(q) > 1` 루프 진입 안 함.

## 2. N = 2

```
입력: 2
출력: 2
```
카드 1 버리고 → 카드 2 남음. 한 라운드만 수행.

## 3. N = 3

```
입력: 3
출력: 2
```
1 버림 → 2를 아래로 → [3, 2]
3 버림 → 2 남음.

## 4. N이 2의 거듭제곱 (N = 4, 8, 16, ...)

답이 항상 N 자체가 된다.
- N=4 → 4
- N=8 → 8

## 5. N = 500,000 (최대 입력)

- deque 사용 시 $O(N)$으로 통과
- list.pop(0) 사용 시 TLE

## 6. 홀수 N

```
입력: 7
출력: 6
```
패턴 확인용.

## 핵심 주의점

- **N = 1 처리**: 루프 진입 조건 `len(q) > 1`이면 자동 처리
- **자료구조 선택**: `collections.deque` 필수 (list의 pop(0)은 $O(N)$)
- **초기화**: `deque(range(1, n+1))`로 1부터 N까지
