# Core-02-Stack-Queue

## 트랙 소개

LIFO/FIFO/덱의 차이를 문제 규칙과 연결해, 자료구조 이름이 아니라 동작 모델로 이해하게 하는 트랙이다.

## 왜 이 순서로 배우는가

스택과 큐는 쉬워 보여도 명령 해석, 예외 처리, 출력 형식에서 자주 실수한다. 이 트랙은 그 반복 실수를 빠르게 잡아 준다.
