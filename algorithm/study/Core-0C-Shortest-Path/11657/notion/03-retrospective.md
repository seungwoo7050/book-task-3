# BOJ 11657 — 회고

## 배운 것

벨만-포드의 핵심은 "V-1번이면 충분"과 "V번째 갱신 = 음의 사이클". 이 두 성질의 증명을 이해하면 알고리즘이 자연스럽다.

## 최단 경로 알고리즘 비교

| 알고리즘 | 음의 가중치 | 음의 사이클 | 시간 |
|----------|:-----------:|:-----------:|------|
| BFS | × | × | $O(V+E)$ |
| 다익스트라 | × | × | $O((V+E)\log V)$ |
| 벨만-포드 | ○ | 탐지 | $O(VE)$ |
| 플로이드-워셜 | ○ | 탐지 | $O(V^3)$ |
