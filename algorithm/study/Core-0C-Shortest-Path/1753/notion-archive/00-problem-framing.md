# BOJ 1753 — 최단경로: 다익스트라

## 첫인상

방향 그래프에서 단일 출발점 최단 경로. CLRS Ch 24.3의 다익스트라 알고리즘 정석 구현. 음의 가중치가 없으므로 다익스트라 사용 가능.

## 문제 구조

- $V$개 정점, $E$개 간선 (방향, 양의 가중치)
- 시작점 $K$에서 모든 정점까지의 최단 거리
- 도달 불가 시 `INF` 출력

## 왜 이 문제를 선택했는가

다익스트라의 표준 구현. 우선순위 큐(힙) + lazy deletion 패턴을 익히는 핵심 문제.

## 난이도 평가

Silver. 다익스트라 자체는 교과서적이지만, 힙 기반 구현의 세부가 중요.
