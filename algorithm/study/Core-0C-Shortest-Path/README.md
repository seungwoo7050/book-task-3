# Core-0C-Shortest-Path

가중치 조건에 따라 어떤 최단 경로 알고리즘을 골라야 할까?

## 왜 이 순서인가

최단 경로는 알고리즘 선택이 핵심이다. 음수 간선 유무, 단일 시작점 여부, 출력 요구를 문서로 정리해 두면 이후 응용이 쉬워진다.
