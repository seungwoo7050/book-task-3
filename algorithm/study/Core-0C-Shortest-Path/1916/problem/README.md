# Problem: 최소비용 구하기 (BOJ 1916)

## Problem Statement

$N$개 도시, $M$개 버스. 각 버스는 출발 도시에서 도착 도시로 비용이 있다. 출발 도시에서 도착 도시까지 최소 비용을 구하라.

## Input

- Line 1: $N$ ($1 \le N \le 1\,000$)
- Line 2: $M$ ($1 \le M \le 100\,000$)
- Next $M$ lines: 출발, 도착, 비용
- Last line: 출발 도시, 도착 도시

## Output

출발 도시에서 도착 도시까지의 최소 비용.

## Examples

**Input**:
```
5
8
1 2 2
1 3 3
1 4 1
1 5 10
2 4 2
3 4 1
3 5 1
4 5 3
1 5
```

**Output**:
```
4
```
