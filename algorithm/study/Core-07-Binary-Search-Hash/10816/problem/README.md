# Problem: Number Card 2 (BOJ 10816)

## Problem Statement

Sanggeun has $N$ number cards. Given $M$ query integers, for each query print how many cards have that number.

## Input

- Line 1: $N$ ($1 \le N \le 500\,000$)
- Line 2: $N$ integers on the cards
- Line 3: $M$ ($1 \le M \le 500\,000$)
- Line 4: $M$ query integers

## Output

For each query, print the count of matching cards, separated by spaces.

## Examples

**Input**:
```
10
6 3 2 10 10 10 -10 -10 7 3
8
10 9 -5 2 3 4 5 -10
```

**Output**: `3 0 0 1 2 0 0 2`
