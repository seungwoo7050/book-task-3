# 이진 탐색 / 해시 개념 정리 — 수 찾기

## CLRS 연결
CLRS Ch 11 Hash Tables — 해시 테이블의 기대 $O(1)$ 탐색.
CLRS Ch 2.3 이진 탐색 — 정렬된 배열에서 $O(\log N)$ 탐색.

## 두 가지 풀이
| 방법 | 시간 | 공간 |
|------|------|------|
| `set` (해시) | $O(N + M)$ 평균 | $O(N)$ |
| 이진 탐색 | $O((N+M) \log N)$ | $O(N)$ |

## 해시 기반 풀이 (채택)
Python `set`은 해시 기반 — `in` 연산이 평균 $O(1)$.
가장 간결하고 실전에서도 빠르다.

## 이진 탐색 풀이
`bisect_left`로 정렬된 배열에서 위치를 찾고, 해당 값이 일치하는지 확인.
