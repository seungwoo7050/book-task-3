# Problem: Find Number (BOJ 1920)

## Problem Statement

Given $N$ integers and $M$ query integers, for each query determine whether the integer exists in the $N$ integers. Print 1 if found, 0 otherwise.

## Input

- Line 1: $N$ ($1 \le N \le 100\,000$)
- Line 2: $N$ integers
- Line 3: $M$ ($1 \le M \le 100\,000$)
- Line 4: $M$ query integers

## Output

For each query, print 1 or 0 on its own line.

## Examples

**Input**:
```
5
4 1 5 2 3
5
1 3 7 9 5
```

**Output**:
```
1
1
0
0
1
```
