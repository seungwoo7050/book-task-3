# Core-07-Binary-Search-Hash

## 트랙 한 줄 질문

탐색 대상을 어떻게 재정의해 선형 탐색을 벗어날까?

## 왜 이 순서인가

많은 문제에서 병목은 구현이 아니라 탐색 전략이다. 이 트랙은 '무엇을 탐색 대상으로 볼 것인가'를 바꾸는 감각을 만든다.
