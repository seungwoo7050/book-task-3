# algorithm 종합 과제 답안지

이 트랙의 공식 답은 “별도 capstone이 없다”는 구조 자체다. 알고리즘은 여러 주제를 하나의 제출 프로젝트로 다시 묶기보다, 핵심 트랙과 심화 트랙을 단계적으로 완주하는 쪽이 더 자연스럽고 실제 학습 효과도 좋다.

## 왜 별도 capstone 답이 없는가

- 이 저장소의 알고리즘 영역은 이미 `16개 트랙 + 53개 개별 문제`로 충분히 긴 학습 경로를 갖는다.
- 웹 백엔드처럼 기능을 하나의 제품으로 합치는 방식보다, 트랙별 사고법을 끊기지 않게 이어 가는 것이 더 중요하다.
- 그래서 capstone을 새로 만들면 오히려 “어떤 알고리즘 사고를 연습하는가”가 흐려진다.
