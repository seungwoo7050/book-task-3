# Static Type Checking 노트

## 목적

이 디렉터리는 `static-type-checking`의 현재 공개용 학습 노트입니다.
공개 README가 탐색용 안내문이라면, 이 폴더는 "어떤 오류를 static 단계로 끌어올렸는가"와 "새 환경에서 어떤 순서로 다시 검증할 것인가"를 압축한 현재판입니다.

## 이 버전에서 다루는 것

- 어떤 오류를 static 단계로 끌어올렸는가
- type environment를 어떤 구조로 유지했는가
- function boundary에서 어떤 규칙이 핵심이었는가
- 2026-03-11 기준 재검증 명령과 성공 신호
