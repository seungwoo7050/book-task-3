# Foundations-CSAPP 블로그 트랙

CSAPP(CS:APP 3rd Edition) 기반 5개 프로젝트의 개발 타임라인.
공식 self-study 자산은 local-only 경계로 유지하고, 공개 저장소에는 C/C++ companion과 학습 로그를 담는다.

## 프로젝트 목록

| 프로젝트 | 핵심 주제 | 시리즈 |
|----------|-----------|--------|
| [Data Lab](datalab/) | 비트 퍼즐, 부동소수점 표현 | [→](datalab/00-series-map.md) |
| [Architecture Lab](archlab/) | Y86, SEQ/HCL, ncopy 최적화 | [→](archlab/00-series-map.md) |
| [Bomb Lab](bomblab/) | 역공학, phase 패턴 분석 | [→](bomblab/00-series-map.md) |
| [Attack Lab](attacklab/) | code injection, ROP chain | [→](attacklab/00-series-map.md) |
| [Performance Lab](perflab/) | cache simulator, matrix transpose | [→](perflab/00-series-map.md) |

## 공개 경계

- **공개**: C/C++ companion 코드, 검증 경로, 학습 로그
- **local-only**: 공식 handout, 바이너리, 채점 스크립트, 정답 문자열/payload
