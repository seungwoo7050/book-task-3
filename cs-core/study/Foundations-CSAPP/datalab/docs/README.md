# Data Lab Docs

## Purpose

This folder keeps the public, repository-facing explanation of the migrated Data Lab project.

The goal is to expose the solving logic, boundary cases, and verification flow without turning
the repository into a pasted answer sheet.

## Document Map

- [`concepts/integer-patterns.md`](concepts/integer-patterns.md): repeated ideas behind puzzles 1~10
- [`concepts/float-boundaries.md`](concepts/float-boundaries.md): classification rules and edge cases for puzzles 11~13
- [`references/verification.md`](references/verification.md): commands and current results, including official `dlc` + `btest`

## Spoiler Policy

- explain core reasoning and edge cases
- explain why certain patterns work
- do not paste the full final implementation as documentation

Longer process notes, debugging chronology, and annotated reference logs live in `../notion/`.
