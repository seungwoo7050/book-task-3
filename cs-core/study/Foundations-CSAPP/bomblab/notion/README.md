# Bomb Lab 노트

## 목적

이 디렉터리는 `bomblab`의 현재 Notion 업로드본입니다.
`../notion-archive/`가 전체 작업 흔적을 보존한다면, 이 폴더는 다시 복습하고 재현할 때 필요한 판단과 순서를 앞쪽으로 끌어온 현재판입니다.

## 이 버전에서 다루는 것

- 공식 bomb와 companion mini-bomb의 역할을 어떻게 나눴는가
- phase를 답이 아니라 패턴으로 읽기 위해 어떤 순서를 썼는가
- 어디서 자주 해석을 틀렸는가
- 새 환경에서 어떤 순서로 다시 검증하면 되는가
- 공개 가능한 설명과 공개하지 않을 정보의 경계는 무엇인가
