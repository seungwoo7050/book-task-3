# Architecture Lab 노트

## 목적

이 디렉터리는 `archlab`의 현재 Notion 업로드본입니다.
`../notion-archive/`가 전체 작업 흔적을 보존한다면, 이 폴더는 다시 복습하고 재현할 때 필요한 판단과 순서를 앞쪽으로 끌어온 현재판입니다.

## 이 버전에서 다루는 것

- Part A/B/C를 어떤 순서로 나눠 이해했는가
- 공식 hand-in 산출물과 companion model을 어떻게 병행 검증했는가
- `iaddq`와 pipeline 비용 모델을 어떻게 연결했는가
- 새 환경에서 어떤 순서로 다시 검증하면 되는가
- 성능 수치를 어떤 문맥에서 읽어야 하는가
