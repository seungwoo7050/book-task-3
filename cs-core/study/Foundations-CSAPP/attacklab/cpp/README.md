# C++ Track

## Scope

This directory contains the fresh C++ companion implementation for Attack Lab.

It mirrors the C verifier but uses the C++ standard library for parsing and phase validation.

## Layout

```text
cpp/
  README.md
  include/
    mini_attacklab.hpp
  src/
    mini_attacklab.cpp
    main.cpp
  tests/
    test_mini_attacklab.cpp
  data/
    phase1.txt
    phase2.txt
    phase3.txt
    phase4.txt
    phase5.txt
  Makefile
```

## Commands

```bash
cd cpp
make clean && make test
```
