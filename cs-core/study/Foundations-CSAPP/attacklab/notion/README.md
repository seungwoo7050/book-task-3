# Attack Lab 노트

## 목적

이 디렉터리는 `attacklab`의 현재 Notion 업로드본입니다.
`../notion-archive/`가 전체 작업 흔적을 보존한다면, 이 폴더는 다시 복습하고 재현할 때 필요한 판단과 순서를 앞쪽으로 끌어온 현재판입니다.

## 이 버전에서 다루는 것

- 문제를 exploit dump가 아니라 payload model로 읽는 방법
- 어떤 순서로 phase를 분리해서 검증했는가
- 실제로 어떤 주소 계산과 gadget 연결에서 흔들리기 쉬운가
- 새 환경에서 어떤 순서로 다시 검증하면 되는가
- 공개 가능한 공격 문서의 경계는 무엇인가
