# Scheduling Simulator 노트

## 목적

이 디렉터리는 `scheduling-simulator`의 현재 공개용 학습 노트다. 공개 README가 탐색용 안내문이라면, 이 폴더는 “왜 이런 규칙과 fixture를 택했는가”와 “새 환경에서 어떤 순서로 다시 검증할 것인가”를 압축한 현재판이다.

## 이 버전에서 다루는 것

- scheduler 정책을 어떤 비교 기준으로 읽을지
- RR와 MLFQ에서 deterministic replay를 만들기 위해 무엇을 고정했는지
- convoy fixture와 interactive fixture가 각각 무엇을 드러내는지
- 2026-03-11 기준 재검증 명령과 성공 신호
