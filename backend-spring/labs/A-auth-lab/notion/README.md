# A-auth-lab 학습 노트 가이드

이 폴더는 A-auth-lab의 공개 학습 노트 세트다. README와 docs에서 다 담지 못한 설계 결정, 디버깅, 재현 과정을 여기에 모아 둔다.

## 먼저 읽을 문서

1. [00-problem-framing.md](00-problem-framing.md): 이 랩이 왜 존재하는지와 성공 기준
2. [05-development-timeline.md](05-development-timeline.md): 처음부터 다시 만들고 싶을 때 따라갈 순서
3. [01-approach-log.md](01-approach-log.md): refresh rotation과 cookie/CSRF 경계를 왜 이렇게 잡았는지

## 목적별 읽기

- 디버깅 기록이 궁금하면 [02-debug-log.md](02-debug-log.md)
- 핵심 개념만 빠르게 복습하려면 [04-knowledge-index.md](04-knowledge-index.md)
- 무엇이 아직 약한지 보려면 [03-retrospective.md](03-retrospective.md)

## 문서 목록

- `00-problem-framing.md`: 문제 정의와 성공 기준
