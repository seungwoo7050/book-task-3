# Knowledge Index

## Reusable concepts

- Health versus readiness:
  - 프로세스 생존과 실제 요청 준비 상태는 다르다.
- Structured JSON logging:
  - 로그 수집과 검색을 염두에 둔 기계 친화적 형식이다.
- Prometheus-style metrics:
  - 수치형 상태를 scrape 가능한 형태로 노출하는 방식이다.

## Glossary

- trace ID:
  - 하나의 요청 흐름을 로그에서 따라가기 위한 식별자다.
- scrape target:
  - 메트릭 수집기가 주기적으로 읽어 가는 endpoint다.
