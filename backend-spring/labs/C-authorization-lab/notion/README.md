이 폴더는 인증과 인가를 분리해 RBAC, membership, ownership을 다룬 과정을 기록한 공개 학습 노트 세트다.

## 먼저 읽을 문서

1. [00-problem-framing.md](00-problem-framing.md): authorization 랩이 따로 필요한 이유
2. [05-development-timeline.md](05-development-timeline.md): 처음부터 다시 따라 만들 때의 순서
3. [01-approach-log.md](01-approach-log.md): service logic과 method security 사이에서 어떤 판단을 했는지

## 목적별 읽기

- 구현 깊이와 설명 범위의 간극을 보려면 [02-debug-log.md](02-debug-log.md)
