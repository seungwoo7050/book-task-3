# Knowledge Index

## Reusable concepts

- Membership lifecycle:
  - 조직 소속은 즉시 부여보다 invite state를 거치는 편이 안전하다.
- Ownership check:
  - 같은 role이라도 생성자나 소유자에게만 허용되는 규칙을 뜻한다.
- Declarative versus imperative authorization:
  - annotation 중심 정책과 service logic 중심 정책의 차이다.

## Glossary

- RBAC:
  - 역할 이름으로 권한 묶음을 표현하는 방식이다.
- method security:
  - Spring에서 메서드 호출 수준으로 권한 규칙을 거는 방식이다.
