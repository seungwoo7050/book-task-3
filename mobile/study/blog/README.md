# Mobile Study Blog

이 디렉터리는 `mobile/study` 아래의 독립 프로젝트만 골라 다시 쓴 블로그 산출물을 모은다.

## 기존 blog 처리 방식

- `isolate-and-rewrite`
- 이번 범위에는 기존 `mobile/study/blog` 초안이 없어서 실제 격리 이동은 필요하지 않았다.
- 새 문서는 기존 blog 초안을 입력으로 쓰지 않고, 소스코드, `README.md`, `docs/`, 테스트, CLI 재검증만 근거로 삼았다.

## 산출물 형식

- `01-evidence-ledger.md`: 독립 프로젝트 판정, 사용한 근거, chronology ledger
- `02-structure.md`: 글의 중심 질문, 구현 순서, 반드시 넣을 근거
- `10-development-timeline.md`: 최종 블로그 본문
