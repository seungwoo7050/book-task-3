# Realtime Chat — Notion 문서 안내

이 폴더는 `realtime-chat` 프로젝트의 개발 과정과 학습 기록을 담는 로컬 전용 문서 세트다.
Notion으로 그대로 옮길 수 있도록 구성했다.

## 어떤 순서로 읽을까

| 순서 | 파일 | 목적 |
|------|------|------|
| 1 | [00-problem-framing.md](00-problem-framing.md) | 이 프로젝트가 왜 존재하고, 어떤 질문에 답하려 했는지 |
| 2 | [05-development-timeline.md](05-development-timeline.md) | 전체 개발 과정을 시간순으로 따라가기 |
| 3 | [01-approach-log.md](01-approach-log.md) | 설계 결정과 선택지를 깊이 이해하기 |

| 5 | [03-retrospective.md](03-retrospective.md) | 끝나고 나서 무엇을 배웠는지 |
| 6 | [04-knowledge-index.md](04-knowledge-index.md) | 다른 프로젝트와 연결 고리, 재사용 가능한 지식 |
