## 현재 정책

- `study/`는 실제 학습과 재구성 작업의 기준 트리다.
- 공개 표면은 `README -> problem -> solution -> docs -> notion` 순서를 따른다.
- `notion/`은 현재 공개 추적되는 학습 노트 세트다. 개인 메모가 아니라 재현과 회고를 위한 보조 문서다.
- 절대 로컬 경로 예시는 README에 남기지 않는다.
