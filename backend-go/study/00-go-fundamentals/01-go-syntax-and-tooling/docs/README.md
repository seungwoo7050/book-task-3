# Docs

## Overview

이 과제는 Go 문법에 처음 들어오는 학습자가 문자열 처리, slice/map, 함수 분리,
그리고 `go run`/`go test` 루프를 가장 작은 예제로 익히는 데 목적이 있다.

## Why This Project

문자열 정규화와 빈도 계산은 입문자가 `for`, `if`, `map`, `slice`, 함수 반환값을
한 번에 연습하기 좋다. 문제 크기를 의도적으로 작게 둬서 언어 문법과 도구 사용에
집중하도록 만들었다.
