# 05 Portfolio Projects

## 이 트랙이 푸는 문제

- 학습 기록만으로는 채용 관점의 대표작 신호가 약하다.

## 이 트랙의 답

- 기존 학습 자산을 runtime dependency로 가져오지 않고 새로 재소유한 대표 포트폴리오 프로젝트를 둔다.

## 프로젝트 순서

1. [18-workspace-saas-api](18-workspace-saas-api/README.md) : JWT auth, 조직 단위 RBAC, async notification, Redis cache를 한 제품형 API로 묶은 대표 포트폴리오 과제다.

## 졸업 기준

- README, OpenAPI, smoke script만 보고 로컬에서 재현 가능한 대표작을 제시할 수 있어야 한다.
- auth, RBAC, tenant boundary, async notification, cache, observability를 하나의 API에서 설명할 수 있어야 한다.

## 대표 프로젝트

- [18-workspace-saas-api](18-workspace-saas-api/README.md) : 채용용으로 직접 보여줄 대표작이다.
