# Docs

## Overview

이 프로젝트는 학습용 capstone과 별도로, 웹 백엔드 공고에 바로 제출할 수 있는
대표 포트폴리오용 B2B SaaS API를 만든다.

## Why This Project

대표작의 기준을 “복잡한 기술 이름”이 아니라, auth/RBAC/tenant boundary/async
notification/cache/reproducibility를 한 API에서 일관되게 보여 주는 것으로 잡았다.
