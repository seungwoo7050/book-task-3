# Docs

## Overview

이 과제는 Go 코드가 아니라 배포 자산을 중심에 둔 infra 과제다.
Docker image build, Helm chart lint/render, ArgoCD manifest를 하나의 GitOps 흐름으로 본다.

## Why This Project

주니어 후반 수준에서는 애플리케이션 코드뿐 아니라 “어떻게 배포되는가”까지
기본 그림을 읽을 수 있어야 한다. 이 과제는 그 최소 범위를 다룬다.
