# 04 Capstone

## 이 트랙이 푸는 문제

- 분리 학습한 기술을 하나의 제품형 흐름으로 다시 조합하지 않으면 제출 가능한 프로젝트로 이어지지 않는다.

## 이 트랙의 답

- 거래 일관성, outbox, 운영 기본 요소를 하나의 게임 상점 API로 통합한 capstone을 둔다.

## 프로젝트 순서

1. [17-game-store-capstone](17-game-store-capstone/README.md) : 거래 일관성, outbox, 운영 기본 요소를 하나의 게임 상점 API로 통합한 필수 capstone이다.

## 졸업 기준

- README만 보고 실행, 테스트, 핵심 시나리오 검증이 가능해야 한다.
- 트랜잭션, 멱등성, outbox를 하나의 서비스 흐름으로 연결해 설명할 수 있어야 한다.
