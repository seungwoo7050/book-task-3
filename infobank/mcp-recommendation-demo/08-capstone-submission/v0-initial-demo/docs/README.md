# v0 Initial Demo Docs

This folder is the stable index for tracked evidence. Long process logs stay in `notion/`.

## Commands

```bash
pnpm install
pnpm db:up
pnpm migrate
pnpm seed
pnpm dev
pnpm test
pnpm eval
pnpm capture:presentation
pnpm e2e
```
