# v1 문서 안내

`v1` 문서는 baseline 대비 무엇이 더 단단해졌는지 보여 준다. 회귀 검증 결과, lineage 정리, fallback 변화, 운영 경로 점검을 함께 읽는 용도다.

## 포함된 문서

- `docs/demo/demo-runbook.md`: v1 시연 순서
- `docs/demo/phase1-vs-phase2-diff-matrix.md`: baseline 대비 변경점 요약
- `docs/demo/demo-fallbacks.md`: dependency fallback 변화
- `docs/demo/proof-artifacts/`: regression proof와 화면 캡처
- `docs/presentation/v1-demo-presentation.md`: 발표용 문서
