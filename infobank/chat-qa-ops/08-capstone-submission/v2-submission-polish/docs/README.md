# v2 문서 안내

`v2` 문서는 최종 제출에 필요한 증빙을 한 번에 모아 둔 stable index다. runbook, compare 결과, improvement report, 발표 자료를 빠르게 찾는 용도로 먼저 본다.

## 포함된 문서

- `docs/demo/demo-runbook.md`: 최종 시연 순서
- `docs/demo/proof-artifacts/`: compare 결과, improvement report, API/CLI 증빙
- `docs/demo/demo-fallbacks.md`: known limits와 fallback 메모
- `docs/presentation/v2-demo-presentation.md`: 발표용 문서
- `docs/demo/phase2-demo-scenario.md`: 최종 시연 시나리오
