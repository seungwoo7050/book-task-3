# v0 문서 안내

`v0` 문서는 "처음 실행 가능한 baseline이 어떤 모습이어야 하는가"를 설명한다. 빠르게 시연 순서를 확인하고, proof artifact가 어디에 쌓이는지 찾는 용도로 먼저 본다.

## 포함된 문서

- `docs/demo/demo-runbook.md`: 5분 시연 순서
- `docs/demo/phase1-demo-proof.md`: seeded session과 평가 결과 설명
- `docs/demo/demo-fallbacks.md`: fallback path와 known limits
- `docs/demo/proof-artifacts/`: 재생성된 API 응답, CLI 결과, 스크린샷
- `docs/presentation/v0-demo-presentation.md`: 발표용 문서
