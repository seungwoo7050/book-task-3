# v3 self-hosted OSS 문서 안내

`v3-self-hosted-oss` 문서는 단순 데모 증빙보다 "다른 팀이 직접 설치해 운영할 수 있는가"를 설명하는 데 초점을 둔다. quickstart, import contract, worker/job 운영 흐름, OSS 공개 메모를 먼저 확인하면 된다.

## 포함된 문서

- `docs/install.md`: Docker Compose와 로컬 설치 경로
- `docs/operations.md`: worker/job 운영 흐름과 기본 운영 절차
- `docs/security-model.md`: auth, 권한, boundary 설명
- `docs/api.md`: 주요 HTTP contract 요약
- `docs/backup-restore.md`: 백업/복구 경로
- `docs/oss-positioning.md`: OSS로 공개할 때의 범위와 기대 사용자
