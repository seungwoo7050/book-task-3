# 아카이브 안내

이 폴더는 현재 `notion/`으로 승격되기 전의 장문 기록, 초기 에세이, 재현 로그를 보존하는 아카이브입니다.

## 무엇이 들어 있는가

- `essay.md`: 당시의 설명형 기록과 긴 서사형 메모를 담습니다.
- `timeline.md`: 당시의 구현 순서, 실행 명령, 검증 흐름을 시간순으로 보존합니다.

## 언제 여는가

- 현재 `notion/`만으로는 판단 배경이 부족할 때 엽니다.
- 재구성 이전 표현이나 구조를 비교해야 할 때 엽니다.
- provenance나 재현 기록이 필요할 때 참고합니다.

현재 기준의 읽기 입구는 `../README.md`와 `../notion/README.md`입니다.
