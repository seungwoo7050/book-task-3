# Docs Index

이 디렉터리는 언어별 루트 규칙과 커리큘럼 재설계 근거를 담는다.

## 문서 지도

- [curriculum-audit.md](curriculum-audit.md): 레거시 프로젝트셋 평가와 language-first 재구성 근거
- [curriculum-map.md](curriculum-map.md): 레거시 모듈과 Go/Python 프로젝트 대응표
- [language-crosswalk.md](language-crosswalk.md): Python 입문 트랙과 Go 정본 트랙의 교차 매핑
- [migration-conventions.md](migration-conventions.md): 디렉터리 모양, 상태 어휘, provenance, 검증 규칙

## 사용 순서

1. 커리큘럼 감사로 왜 Python-first, Go-second 구성을 택했는지 읽는다.
2. 커리큘럼 맵으로 레거시와 새 프로젝트의 대응을 본다.
3. 교차 매핑으로 Python 프로젝트가 Go의 어느 범위를 접는지 확인한다.
4. 마이그레이션 규칙으로 각 언어 프로젝트의 공개 표면과 검증 방식을 확인한다.
