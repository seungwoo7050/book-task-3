# Catalog Guide

전체 프로젝트를 빠르게 훑고 싶다면 [project-catalog.md](project-catalog.md)를 먼저 여세요. 각 행에서 바로 `문제`, `내 해법`, `검증 명령`, `상태`를 확인할 수 있습니다.

## 사용하는 법

- 언어를 먼저 고르고 싶다면 `언어` 컬럼을 봅니다.
- 개념 흐름을 따라가고 싶다면 `트랙`과 `프로젝트` 컬럼을 왼쪽에서 오른쪽으로 읽습니다.
- 특정 구현 방식을 찾고 싶다면 `내 해법` 컬럼을 기준으로 훑습니다.
- 실행 명령이 궁금하면 `검증 명령` 컬럼에서 바로 확인합니다.
