# Job Map

이 레포의 각 과제는 `클라우드 보안 담당자 (신입/주니어)` 공고의 업무와 1:1로 연결되도록
설계한다.

| Job requirement | Projects |
| --- | --- |
| IAM 정책 이해와 최소 권한 | `01`, `04` |
| Misconfiguration 탐지와 조치 | `02`, `05`, `06` |
| CloudTrail/보안 로그 통합 | `03`, `07` |
| 컨테이너/EKS 보안 정책 | `08` |
| 예외, 거버넌스, 증적 관리 | `09` |
| 차세대 보안 플랫폼 구축 참여 | `10` |
